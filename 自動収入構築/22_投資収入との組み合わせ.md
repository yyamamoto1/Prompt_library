# 投資収入との組み合わせ戦略

あなたは副業収入と投資を組み合わせた資産形成の専門家です。

以下の情報をもとに、**事業収入と投資収入を組み合わせた資産形成戦略**を立ててください。

- 事業収入を投資で増やす

💹 **事業収入×投資収入の戦略**

**収入ポートフォリオの考え方**

```
【経済的自由への道筋】

事業収入（能動的収入）
    ↓ 一部を投資へ
投資収入（受動的収入）
    ↓ 複利で成長
経済的自由（生活費 < 不労所得）

【ステージ別の目標】

Stage 1：事業収入の安定化
・月収30万円以上の安定
・生活費を差し引いて余剰を作る

Stage 2：投資の開始
・余剰資金を投資に回す
・月5-10万円からスタート

Stage 3：資産の成長
・複利効果で資産が増加
・配当/分配金が入り始める

Stage 4：経済的自由
・投資収入が生活費を上回る
・働くかどうかを選べる状態
```

**事業と投資のバランス**

```
【フェーズ別の重点】

■ 事業立ち上げ期
事業：90% / 投資：10%
・事業収入の最大化が優先
・余剰があれば少額から投資

■ 事業安定期
事業：70% / 投資：30%
・事業の仕組み化を進める
・投資を本格的にスタート

■ 事業成熟期
事業：50% / 投資：50%
・事業は維持・効率化
・投資収入を増やす

■ 経済的自由達成後
事業：選択 / 投資：生活費以上
・働くかどうかは自由
・投資収入で生活可能

【注意点】
・事業への投資（成長）も重要
・事業が不安定なうちは投資は控えめに
・ハイリスク投資は事業が安定してから
```
