**サブスクリプションの類型分析**

```
【サブスクモデルの種類】

① アクセス型
   例：Netflix、Spotify
   価値：コンテンツへの無制限アクセス
   課題：常に新コンテンツが必要

② キュレーション型
   例：雑誌定期購読、BOXサービス
   価値：厳選されたものが届く
   課題：選定眼・センスが重要

③ 補充型
   例：Amazon定期おトク便、消耗品
   価値：買い忘れ防止、割引

④ サービス型
   例：Adobe CC、Microsoft 365
   価値：ツール・機能の継続利用
   課題：継続的な価値提供

⑤ コミュニティ型
   例：オンラインサロン
   価値：つながり、情報
   課題：コミュニティ運営力
```

**価格戦略**

```
【価格設定の考え方】

① コスト基準
   原価 + 利益 = 価格

② 競合基準
   競合価格 ± 差別化分 = 価格

③ 価値基準（推奨）
   顧客が感じる価値 = 価格の上限

【価格帯別の特徴】

月額1,000円以下：
・衝動的に登録しやすい
・解約もされやすい
・大量の会員が必要

月額1,000-5,000円：
・バランスが良い価格帯
・ある程度のコミットメント
・中規模の会員数で成立

月額5,000-10,000円：
・高い価値提供が必要
・解約されにくい
・少数精鋭で成立

月額10,000円以上：
・プレミアムサービス
・濃いサポートが必要
・ハイタッチが求められる

【推奨価格】
月額[X]円
根拠：[根拠]
```

**解約防止（チャーン対策）**

```
【解約の主な理由と対策】

理由1：価値を感じなくなった
対策：
・オンボーディングの強化
・利用状況に応じたリマインド
・成功事例の共有

理由2：使わなくなった
対策：
・利用促進メール
・新機能のお知らせ
・活用ガイドの提供

理由3：競合に乗り換え
対策：
・差別化ポイントの強化
・スイッチングコストの設計
・ロイヤリティ特典

理由4：経済的理由
対策：
・ダウングレードオプション
・一時休止プラン
・年間プラン割引

【解約フロー設計】
解約ボタン
  ↓
解約理由アンケート
  ↓
オファー提示（割引/ダウングレード）
  ↓
解約確認
```
