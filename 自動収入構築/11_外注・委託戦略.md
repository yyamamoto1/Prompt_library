# 外注・業務委託戦略

あなたはビジネスの外注化・委託戦略の専門家です。

以下の情報をもとに、**自分の時間を使わずに収益を生む外注戦略**を立ててください。

【目的】
- 自分の時間を使わずに事業を回す
- スケーラブルなビジネスモデル構築
- 時間単価の最大化

【入力】
- 現在の事業内容：[例：ブログ運営、コンテンツ販売]
- 月間作業時間：[例：40時間]
- 外注可能な予算：[例：月5万円]
- 外注したい業務：[例：記事作成、デザイン]
- 目標：[例：作業時間を半分に減らしつつ収益維持]

【出力形式】

**外注費用と効果の計算**

```
【投資対効果シミュレーション】

■ 外注前
自分の作業時間：40時間/月
自分の時給換算：3,000円
時間コスト：120,000円相当

■ 外注後
自分の作業時間：20時間/月
外注費用：50,000円/月
浮いた時間：20時間/月

■ 効果
浮いた時間の価値：20h × 3,000円 = 60,000円
外注費用：50,000円
純利益：10,000円 + 時間的自由

■ 浮いた時間の活用
・新規事業の立ち上げ
・高単価業務への集中
・自己投資
・プライベート
```

**外注管理のベストプラクティス**

```
【品質管理】
□ 最初は小さな案件でテスト
□ フィードバックは具体的に
□ 良い成果は褒める
□ 定期的なすり合わせ

【リスク管理】
□ 複数の外注先を確保
□ 重要情報の取り扱いルール
□ 契約書/NDAの締結
□ 納期に余裕を持たせる

【関係構築】
□ 適正な報酬を支払う
□ 迅速な支払い
□ 長期的な関係を目指す
□ 感謝を伝える

【効率化】
□ 定型業務はテンプレート化
□ 指示はできるだけ具体的に
□ ツールを活用（Notion、Slack等）
□ 定期MTGで認識合わせ
```
