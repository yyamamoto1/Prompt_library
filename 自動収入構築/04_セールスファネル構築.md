**STEP 4：セールスページ**

```
【セールスページの構成】

① キャッチコピー/ヘッドライン
   → 最大のベネフィットを一言で

② 問題提起
   → 読者の悩みを言語化

③ 共感/ストーリー
   → 「私も同じでした」

④ 解決策の提示
   → この商品で解決できる

⑤ 商品説明
   → 何が含まれているか

⑥ ベネフィット
   → 得られる結果

⑦ 実績/お客様の声
   → 社会的証明

⑧ 価格/オファー
   → 価値を感じさせてから価格

⑨ 保証
   → リスクリバーサル

⑩ CTA（行動喚起）
   → 今すぐ購入ボタン

⑪ FAQ
   → よくある質問

⑫ 追伸
   → 最後の一押し
```

**STEP 5：自動化の実装**

```
【ツール構成例】

[集客] → [LP] → [メール配信] → [決済] → [商品提供]

WordPress → ペライチ → ConvertKit → Stripe → 会員サイト

【自動化のポイント】

✅ メール配信の自動化
  ・登録と同時にステップメール開始
  ・タグ付けで配信を分岐

✅ 決済の自動化
  ・購入と同時にお礼メール
  ・購入者リストへ自動追加

✅ 商品提供の自動化
  ・デジタル商品なら即時アクセス
  ・会員サイトへの自動招待

✅ フォローアップの自動化
  ・購入後のサンキューメール
  ・X日後のフォローメール
```
