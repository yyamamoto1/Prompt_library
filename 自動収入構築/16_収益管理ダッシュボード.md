**データ収集の自動化**

```
【収入源別のデータ取得方法】

■ 広告収入（Googleアドセンス）
取得方法：AdSense API / 自動レポート
頻度：日次
自動化：Googleスプレッドシートに自動連携

■ アフィリエイト収入
取得方法：ASP管理画面 / API（一部）
頻度：日次〜週次
自動化：スクレイピング or 手動（月1回）

■ 商品販売（Stripe/PayPal）
取得方法：API / Webhook
頻度：リアルタイム
自動化：Zapier経由でスプレッドシートに記録

■ サブスク収入
取得方法：決済サービスAPI
頻度：日次
自動化：自動でMRR計算

■ ECサイト（STORES/BASE等）
取得方法：管理画面 / API
頻度：日次
自動化：CSVエクスポート→自動取り込み
```

**自動化のフロー**

```
【データ収集の自動化】

[各収入源] → [Zapier/Make] → [スプレッドシート]
                               ↓
                        [自動集計・グラフ化]
                               ↓
                        [Looker Studio表示]
                               ↓
                        [週次レポート自動送信]

【設定例】
1. Stripe決済 → Zapierがトリガー
2. スプレッドシートに行を追加
3. 関数で自動集計
4. グラフが自動更新
5. 週1回サマリーメールを自動送信
```
