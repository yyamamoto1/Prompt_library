**青色申告のメリット**

```
【青色申告で受けられる特典】

① 青色申告特別控除
・65万円控除（e-Tax + 複式簿記）
・55万円控除（複式簿記のみ）
・10万円控除（簡易簿記）

② 赤字の繰越控除
・赤字を3年間繰り越せる
・将来の黒字と相殺可能

③ 青色事業専従者給与
・家族への給与を経費にできる

④ 少額減価償却資産の特例
・30万円未満の資産を一括経費化

【青色申告の始め方】
1. 開業届を提出（開業から1ヶ月以内）
2. 青色申告承認申請書を提出
   ・1/1〜1/15開業：その年の3/15まで
   ・1/16以降開業：開業から2ヶ月以内
3. 帳簿をつける（会計ソフト推奨）
4. 確定申告時に青色申告決算書を提出
```

**節税対策の具体策**

```
【すぐにできる節税】

① 経費を漏れなく計上
□ レシート/領収書を必ず保管
□ クレジットカードを事業用に分ける
□ 経費管理アプリで記録

② 青色申告65万円控除
□ 開業届 + 青色申告承認申請
□ 会計ソフトで複式簿記
□ e-Taxで電子申告

③ 小規模企業共済
□ 掛金が全額所得控除
□ 月1,000円〜70,000円
□ 退職金代わりになる

④ iDeCo（個人型確定拠出年金）
□ 掛金が全額所得控除
□ 運用益も非課税
□ 老後資金を積み立て

【節税効果シミュレーション】
所得：200万円の場合

何もしない場合の税金：約30万円

節税後：
・青色65万円控除：▲約10万円
・小規模企業共済（月2万円）：▲約3.5万円
・iDeCo（月2.3万円）：▲約4万円
→ 税金：約12.5万円（17.5万円の節税）
```

**確定申告の準備**

```
【年間スケジュール】

1-3月：確定申告
・2/16〜3/15が申告期間
・e-Taxなら1月から可能

4-12月：日々の記帳
・収入と経費を記録
・領収書を整理

12月：年末調整（会社員の場合）
・副業収入は含まれない

1月：年間の集計
・収入合計
・経費合計
・所得の計算

【必要な書類】
□ 収入がわかる資料
  ・ASPの報酬明細
  ・決済サービスの取引履歴
  ・銀行の入金記録

□ 経費の証拠
  ・領収書/レシート
  ・クレジットカード明細
  ・銀行引き落とし記録

□ 控除の証明書
  ・生命保険料控除証明書
  ・小規模企業共済掛金証明書
  ・iDeCo掛金証明書
```
