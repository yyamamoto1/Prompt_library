# リスク分散・事業継続計画

あなたは事業リスク管理の専門家です。

以下の情報をもとに、**収入源のリスクを分散し、事業を継続するための計画**を立ててください。

【目的】
- 収入源のリスク特定と対策
- 事業継続のための備え
- 突発的な収入減への対応

【入力】
- 主な収入源：[例：ブログ広告、アフィリエイト、商品販売]
- 依存しているプラットフォーム：[例：Google、Amazon、特定ASP]
- 過去に経験したリスク：[例：アルゴリズム変更で収入半減]
- 月間収入：[例：50万円]

【出力形式】

**リスク別対策**

```
【プラットフォーム依存リスク】

現状：Google検索に80%依存

対策1：トラフィック源の分散
・SNS（Twitter、Instagram）からの流入強化
・YouTube開設
・メルマガリストの構築
→ Google依存度を50%以下に

対策2：自社メディアの強化
・メールリストの構築（最重要）
・LINEリストの構築
→ プラットフォームに左右されない顧客基盤

対策3：複数ドメイン運営
・リスク分散のため複数サイト
→ 1つがペナルティを受けても他でカバー

---

【収入源リスク】

現状：アフィリエイト収入に60%依存

対策1：収入源の多様化
・自社商品の開発
・サブスク収入の構築
・広告収入の追加
→ 1つの収入源は40%以下に

対策2：複数ASPの利用
・A8、もしも、バリューコマース等
・同じ案件でも複数経由で

対策3：継続報酬型へのシフト
・単発成果報酬 → 継続報酬
・サーバー、ツール系を強化

---

【個人リスク】

対策1：自動化の推進
・作業しなくても収入が続く仕組み
・コンテンツ資産の積み上げ

対策2：外注チームの構築
・自分がいなくても回る体制
・マニュアルの整備

対策3：保険・貯蓄
・収入保障保険の検討
・6ヶ月分の生活費を確保
```

**事業継続計画（BCP）**

```
【シナリオ別対応計画】

■ シナリオ1：検索順位大幅下落
━━━━━━━━━━━━━━━━━━━━
影響：月収50万円 → 25万円
対応：
・SNS/メルマガでの集客強化（即時）
・広告出稿の検討（1週間以内）
・コンテンツ改善（1ヶ月以内）
・新規サイト構築（3ヶ月以内）

■ シナリオ2：主要案件の終了
━━━━━━━━━━━━━━━━━━━━
影響：月収50万円 → 35万円
対応：
・代替案件への差し替え（即時）
・新規案件の開拓（1週間以内）
・自社商品での補填（1ヶ月以内）

■ シナリオ3：病気で1ヶ月稼働停止
━━━━━━━━━━━━━━━━━━━━
影響：新規収入なし（ストック収入のみ）
対応：
・予約投稿の設定（事前準備）
・外注への引き継ぎ（事前準備）
・貯蓄からの補填
```

**緊急時チェックリスト**

```
【収入急減時の対応】

□ 現状の把握
  ・何がどれだけ減ったか
  ・原因は何か

□ 短期対策（1週間以内）
  ・他の収入源でカバー可能か
  ・緊急で増やせる施策はあるか
  ・支出の削減

□ 中期対策（1-3ヶ月）
  ・根本原因への対処
  ・新規収入源の開拓
  ・ビジネスモデルの見直し

□ 長期対策（3ヶ月以降）
  ・再発防止策
  ・さらなる分散化
  ・事業継続計画の更新
```
