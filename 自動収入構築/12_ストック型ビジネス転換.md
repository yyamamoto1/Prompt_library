# フロー型からストック型ビジネスへの転換

あなたはビジネスモデル転換の専門家です。

以下の情報をもとに、**労働集約型（フロー）から資産型（ストック）への転換戦略**を立ててください。

【目的】
- 時間を切り売りするビジネスからの脱却
- 継続的に収益を生む資産の構築
- 労働時間に比例しない収入の実現

🔄 **フロー→ストック転換戦略**

**ビジネスモデルの違いを理解する**

```
【フロー型ビジネス】
収入 = 労働時間 × 時間単価

特徴：
・働いた分だけ収入
・時間の上限が収入の上限
・休むと収入ゼロ
・毎月ゼロからスタート

例：
・受託制作
・コンサルティング
・講師/セミナー
・サービス提供

【ストック型ビジネス】
収入 = 資産 × 収益率

特徴：
・資産が収益を生む
・時間に比例しない
・休んでも収入継続
・積み上がっていく

例：
・コンテンツ販売
・サブスクリプション
・ライセンス収入
・広告収入
```

**転換パターンと実例**

```
【パターン1】1対1 → 1対多

Before：個別コンサル（1時間1万円）
After：オンライン講座（1万円×無限）

転換方法：
・コンサル内容を動画化
・よくある質問をFAQ化
・Q&Aはコミュニティで対応

【パターン2】受託 → テンプレート販売

Before：Webサイト制作（1件30万円）
After：テンプレート販売（3万円×無限）

転換方法：
・汎用的なテンプレートを作成
・カスタマイズガイドを付属
・サポートはマニュアルで対応

【パターン3】単発 → 継続課金

Before：単発セミナー（5,000円×30人）
After：月額サロン（3,000円×100人）

転換方法：
・セミナー内容をアーカイブ化
・月額で見放題に
・コミュニティ価値を追加

【パターン4】時間課金 → 成果課金

Before：時給制のアドバイス
After：成果報酬/ロイヤリティ

転換方法：
・クライアントの成果に連動
・長期的な関係を構築
・複数クライアントから継続収入
```
