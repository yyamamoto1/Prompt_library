# メンバーシップサイト構築

あなたはメンバーシップビジネスの専門家です。

以下の情報をもとに、**継続課金で安定収入を得るメンバーシップサイト**を設計してください。

【目的】
- 毎月の安定した継続収入
- 顧客との長期的な関係構築
- スケーラブルなビジネスモデル

【入力】
- 提供するテーマ：[例：副業ノウハウ、プログラミング学習]
- ターゲット：[例：副業を始めたい会社員]
- 想定月額料金：[例：3,000円]
- 提供できるコンテンツ：[例：動画、記事、ライブ配信]
- 目標会員数：[例：100人]

【出力形式】

**会員維持（継続率向上）戦略**

```
【解約を防ぐ施策】

① オンボーディングの充実
   ・登録直後のウェルカムメール
   ・最初にやるべきことを明確に
   ・30日間の活用ガイド

② 定期的な接点
   ・週1回のメール配信
   ・月1回のライブ配信
   ・コミュニティでの声かけ

③ 成功体験の創出
   ・小さな成果を称える
   ・成功事例の共有
   ・進捗の可視化

④ 継続特典
   ・継続月数に応じた特典
   ・長期会員限定コンテンツ
   ・ロイヤリティプログラム

【継続率の目標】
月間解約率：5%以下
年間継続率：60%以上
```
