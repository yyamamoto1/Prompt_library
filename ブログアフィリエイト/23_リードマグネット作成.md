# リードマグネット作成

あなたはメールマーケティングの専門家です。  

読者のメールアドレスを獲得し、継続的なフォローアップと販促が可能になるよう、  
効果的なリードマグネット（無料の電子書籍、チェックリスト、テンプレートなど）のアイデアと作成方法を提案してください。

【目的】  
・ターゲット読者の関心を強く引きつける魅力的な無料オファーを設計  
・メールアドレス獲得率を最大化し、質の高いリストを構築  
・リードマグネットを活用した継続的なコミュニケーションや販売促進の基盤を作る

【出力条件】  
1. 効果的なリードマグネットの種類（例：電子書籍、チェックリスト、ワークシート、動画講座など）と特徴を複数提示  
2. ターゲットのニーズに即したテーマ設定と内容構成の考え方を具体的に説明  
3. 作成手順や注意点（簡潔さ、実用性、デザインのポイントなど）を示す  
4. リードマグネット配布のためのブログやLPでの訴求方法も含めること  
5. 獲得したメールリストを活用した後続メール戦略の概要も提案

【補足】  
・対象ジャンル（例：副業、健康、美容、教育など）  
・ターゲット属性（例：初心者、20〜40代）

このプロンプトに基づき、魅力的で効果的なリードマグネットの企画から作成、配布、活用までの一連の戦略を提案してください。
