# クロスプラットフォーム展開戦略

あなたはSNSマーケティングとYouTube運営の専門家です。

以下の情報をもとに、**YouTube動画を他SNSに展開してリーチを最大化する戦略**を設計してください。

【目的】
- YouTube動画への流入増加
- 複数プラットフォームでのアフィリエイト展開
- ブランド認知の拡大

【入力】
- YouTubeチャンネルジャンル：[例：美容・スキンケア]
- 現在のSNSアカウント：[例：Instagram、Twitter]
- 動画の種類：[例：レビュー、How To、Vlog]

【出力形式】

---
🌐 **クロスプラットフォーム戦略**

**プラットフォーム別役割分担**

| プラットフォーム | 役割 | 投稿頻度 |
|------------------|------|----------|
| YouTube（本編） | メインコンテンツ | 週2本 |
| YouTubeショート | 認知拡大・誘導 | 毎日 |
| Instagram（リール）| 若年層獲得 | 毎日 |
| Instagram（フィード）| ブランディング | 週3回 |
| TikTok | バズ狙い・認知 | 毎日 |
| Twitter/X | 告知・エンゲージ | 毎日 |

**YouTube→各SNSへのリパーパス**

【YouTubeショート】
- 本編の見どころを60秒に編集
- 「続きはチャンネルで」で誘導
- 縦型に再編集

【Instagram リール】
- YouTubeショートを転用
- Instagramトレンド音源に差し替え
- プロフィールリンクへ誘導

【TikTok】
- 最もバズりやすい部分を切り出し
- TikTokトレンドに合わせた編集
- コメント欄でYouTube誘導

【Twitter/X】
- 動画公開告知ツイート
- 動画の要点を箇条書き
- 切り抜き動画投稿

**各プラットフォームでのアフィリエイト展開**

| プラットフォーム | アフィリエイト方法 |
|------------------|-------------------|
| YouTube | 概要欄リンク |
| Instagram | プロフィールリンク、ストーリーズリンク |
| TikTok | プロフィールリンク |
| Twitter | ツイート内リンク |

**投稿スケジュール例（1動画あたり）**

```
Day 0：YouTube本編公開
Day 0：Twitter告知
Day 1：YouTubeショート投稿
Day 1：Instagram リール投稿
Day 2：TikTok投稿
Day 3：Twitter切り抜き投稿
Day 7：Instagram フィード投稿（まとめ）
```
