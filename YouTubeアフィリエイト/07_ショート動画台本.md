# YouTubeショート動画台本生成

あなたはYouTubeショートでバズる動画を量産するプロクリエイターです。

以下の情報をもとに、**60秒以内で視聴者を惹きつけ、本編やアフィリエイトリンクへ誘導するショート動画台本**を5パターン作成してください。

【目的】
- 最初の1秒で視聴者を掴む
- 最後まで見てもらう（視聴完了率UP）
- 本編動画やプロフィールリンクへの誘導

【入力】
- テーマ：[例：副業で稼ぐコツ]
- 紹介したい商品/サービス：[例：副業スクール]
- 本編動画との関連：[例：詳細は長尺動画で解説]

【出力形式】

---
📱 **ショート動画台本（5パターン）**

**パターン①：衝撃事実型（60秒）**
```
0-2秒：「99%の人が知らない事実」
3-15秒：[衝撃的な情報を提示]
16-40秒：[具体的な解説]
41-55秒：[結論・価値提供]
56-60秒：「詳しくはプロフィールから」
```

**パターン②：ビフォーアフター型（60秒）**
```
0-2秒：「1ヶ月前の私」
3-20秒：[ビフォーの状態を見せる]
21-40秒：[やったこと・プロセス]
41-55秒：「今の私」[アフターを見せる]
56-60秒：「方法はプロフィールリンクから」
```

**パターン③：ランキング型（60秒）**
```
0-2秒：「〇〇TOP3」
3-20秒：3位：[内容]
21-40秒：2位：[内容]
41-55秒：1位：[内容]
56-60秒：「他にもあるよ→プロフィール」
```

**パターン④：How To型（60秒）**
```
0-2秒：「〇〇のやり方」
3-50秒：[ステップバイステップで解説]
51-60秒：「もっと詳しく知りたい人は→」
```

**パターン⑤：ストーリー型（60秒）**
```
0-5秒：[問題提起・共感ポイント]
6-40秒：[ストーリー展開]
41-55秒：[解決・結論]
56-60秒：[CTA]
```

---

🎯 **バズらせるコツ**
- テキストテロップは大きく、読みやすく
- BGMは人気楽曲を使用
- 最初の1秒が命
- ループ構造で視聴完了率UP
