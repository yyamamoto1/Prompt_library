# コメント誘導・返信戦略

あなたはYouTubeコミュニティマネジメントのプロです。

以下の情報をもとに、**コメント数を増やし、エンゲージメントを高める戦略**を設計してください。

【目的】
- コメント数増加（アルゴリズム評価向上）
- 視聴者との信頼関係構築
- アフィリエイト商品に関する質問・興味の喚起

【入力】
- チャンネルジャンル：[例：ガジェットレビュー]
- 現在のコメント状況：[例：平均10件/動画]
- ターゲット視聴者：[例：20-40代男性、テック好き]

【出力形式】

---
💬 **コメント誘導戦略**

**動画内でのコメント誘導フレーズ**

① 意見を求める
- 「皆さんはどっち派ですか？コメントで教えてください」
- 「他にもおすすめがあったらコメントで共有してください」

② 質問を投げかける
- 「次にレビューしてほしい商品はありますか？」
- 「実際に使ってる人、感想を聞かせてください」

③ 経験をシェアさせる
- 「同じ経験した人、いいねください」
- 「成功した人は何ヶ月かかった？コメントで教えて」

④ 議論を促す
- 「これには賛否あると思います。皆さんの意見は？」
- 「正直、コスパ的にどう思いますか？」

**アフィリエイト商品への興味喚起**
- 「この商品使ってる人、正直な感想聞かせて」
- 「購入検討中の人、質問あればコメントで」
- 「買うか迷ってる人、背中押してほしい？」

**コメント返信テンプレート**

【質問への回答】
```
[名前]さん、ご質問ありがとうございます！
[回答内容]
他にも気になることがあればお気軽にどうぞ😊
```

【ポジティブコメント】
```
[名前]さん、嬉しいコメントありがとうございます！
[共感や補足]
これからも参考になる動画作りますね💪
```

【購入報告】
```
[名前]さん、購入されたんですね！おめでとうございます🎉
使ってみた感想もぜひ教えてください！
```

【批判・ネガティブコメント】
```
ご意見ありがとうございます。
[建設的な返答]
今後の参考にさせていただきます。
```

**ピン留めコメント戦略**
- 自分のコメント：「質問はこちらに！」
- 視聴者コメント：良い質問や有益な情報をピン留め
