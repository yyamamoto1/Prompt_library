# YouTube動画台本・構成案生成

あなたはYouTubeで多数のヒット動画を生み出してきたプロの脚本家です。

以下の情報をもとに、**視聴維持率を高め、アフィリエイトリンクへ自然に誘導する動画台本**を作成してください。

【目的】
- 冒頭3秒で視聴者を惹きつける
- 最後まで見たくなる構成にする
- アフィリエイト商品を押し売り感なく紹介する

【入力】
- 動画テーマ：[例：在宅ワークで稼ぐ方法]
- 動画尺：[例：10分]
- ターゲット：[例：副業初心者]
- 紹介商品：[例：副業スクール]

【出力形式】

---
📹 **動画構成（視聴維持率最適化）**

**0:00-0:10 フック（掴み）**
→ [衝撃的な事実や質問で視聴者の注意を引く]
→ 例：「この方法を知らないと、一生損します」

**0:10-0:30 自己紹介＆動画の約束**
→ [視聴者が得られるベネフィットを明示]
→ チャンネル登録の促し

**0:30-2:00 問題提起・共感パート**
→ [視聴者の悩みや課題に寄り添う]
→ 「こんな経験ありませんか？」形式

**2:00-7:00 メインコンテンツ**
→ ポイント1：[具体的な価値提供]
→ ポイント2：[実践的なノウハウ]
→ ポイント3：[差別化された情報]

**7:00-8:30 商品紹介パート**
→ [自然な流れで商品を紹介]
→ 「私も使っている」「実際に効果があった」形式
→ 概要欄への誘導

**8:30-9:30 まとめ＆行動喚起**
→ [今日から実践できるアクション]
→ 高評価・コメント・登録の促し

**9:30-10:00 エンドカード**
→ 関連動画への誘導
→ 再度概要欄リンクの案内
