# 終了画面・カード戦略

あなたはYouTubeの機能を最大限活用する動画マーケターです。

以下の情報をもとに、**終了画面とカード機能を使った回遊率・CV最大化戦略**を設計してください。

【目的】
- チャンネル内の回遊率向上
- アフィリエイト関連動画への誘導
- 登録者数の増加

【入力】
- チャンネルの主要動画カテゴリ：[例：レビュー、How To、Vlog]
- アフィリエイト商品紹介動画：[例：〇〇レビュー動画]
- 動画の平均尺：[例：10分]

【出力形式】

---
🎬 **終了画面戦略**

**基本設計（最後20秒）**
```
┌─────────────────────────────┐
│                             │
│   [動画1]        [動画2]    │
│   関連動画       最新動画   │
│                             │
│        [チャンネル登録]     │
│                             │
└─────────────────────────────┘
```

**推奨する終了画面パターン**

① レビュー動画の場合
- 動画1：同カテゴリの別商品レビュー
- 動画2：比較動画
- 目的：購買検討者を逃さない

② How To動画の場合
- 動画1：次のステップの動画
- 動画2：関連ツール/商品レビュー
- 目的：学習意欲を次の動画へ

③ 企画動画の場合
- 動画1：最も人気の動画
- 動画2：関連企画
- 目的：エンタメ→教育系へ誘導

**カード戦略**

【設置タイミング】
- 2分地点：チャンネル登録カード
- 5分地点：関連動画カード（アフィリエイト動画）
- 離脱が多いポイント：救済用動画カード

【カード使用上限】
- 10分動画：3-4枚が最適
- タイミング：視聴維持率が下がり始める前

**セリフとの連携**
```
[7分地点]
「詳しくは右上のカードから見られます」
→ カード表示
```

**アフィリエイト動画への誘導フロー**
```
入口動画（How To）
    ↓ 終了画面
レビュー動画（商品紹介）
    ↓ 概要欄
アフィリエイトリンク
```
