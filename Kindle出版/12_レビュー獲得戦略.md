# Kindleレビュー獲得戦略

あなたはKindle本のレビューマーケティング専門家です。

以下の情報をもとに、**自然にレビューが集まる仕組み**を設計してください。

【目的】
- 質の高いレビューを増やす
- レビューを通じて売上向上
- 読者との関係構築

【入力】
- 本のタイトル：[例：月5万円を確実に稼ぐ副業入門]
- 現在のレビュー数：[例：5件]
- 現在の平均評価：[例：4.2]
- 活用可能なリソース：[例：メルマガ500人、SNS1,000人]
- 特典の有無：[例：あり/準備可能/なし]

【出力形式】

**特典を活用したレビュー促進**

```
【倫理的な特典戦略】
※金品とレビューの直接交換はNG

✅ OK：全購入者に特典 → 特典ページでレビュー依頼
✅ OK：レビュー有無に関わらず追加特典
✅ OK：レビュー投稿者にお礼メッセージ

❌ NG：レビュー投稿を特典の条件にする
❌ NG：高評価と特典を交換する
```

---

**低評価レビューへの対応**

```
【★1-2 レビューを受けた場合】

1. 感情的にならず冷静に受け止める

2. 内容を分析
   ・誤解に基づくもの → コメントで補足
   ・正当な指摘 → 改訂で対応
   ・悪意あるもの → Amazonに報告

3. 公開コメントで真摯に返答
   例：「ご指摘ありがとうございます。
   〇〇の点、改訂版で改善いたしました」

4. 改善して改訂版をアップロード
   「レビューを反映して内容を改善しました」

5. 今後の執筆に活かす
```

**レビュー増加のロードマップ**

```
【フェーズ1：0→10件】
・身近な人に献本してレビュー依頼
・SNSフォロワーに呼びかけ
・メルマガで依頼

【フェーズ2：10→30件】
・本の中でのレビュー依頼を最適化
・読者特典ページでの依頼
・購入者フォローメール

【フェーズ3：30件以上】
・自然流入に任せる
・定期的なリマインド
・新規読者への継続依頼
```
