**KDPでの多言語出版設定**

```
【出版手順】

1. 新規タイトルとして登録
   ・日本語版とは別のタイトルとして作成

2. 言語設定
   ・該当言語を選択

3. マーケットプレイスの選択
   ・英語 → Amazon.com, Amazon.co.uk 等
   ・中国語 → Amazon.cn（特殊な手続き必要）
   ・スペイン語 → Amazon.es, Amazon.com.mx 等

4. 価格設定
   ・現地の相場を調査して設定
   ・為替を考慮

5. カテゴリ・キーワード
   ・現地で検索されるキーワードを調査
```
