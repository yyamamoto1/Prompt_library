**相互送客の仕組み**

```
【本の中での相互紹介】

1. 「はじめに」で既刊の紹介
   「本書は〇〇シリーズの第2巻です。
   第1巻をまだお読みでない方は、
   そちらから読んでいただくとより理解が深まります」

2. 関連する章での誘導
   「この点については、第3巻で詳しく解説しています」

3. 「おわりに」で次巻の予告
   「次巻では〇〇について詳しく解説します。
   ぜひお楽しみに！」

4. 巻末にシリーズ一覧ページ
   全巻のタイトル・概要・購入リンク
```

**出版スケジュール**

```
【推奨ペース】
2〜3ヶ月に1冊

【スケジュール例】
1月：第2巻 執筆
2月：第2巻 出版 → 第1巻とセット割引キャンペーン
3月：第3巻 執筆
4月：第3巻 出版 → シリーズまとめ買いキャンペーン
...

【出版タイミングの最適化】
・前巻のレビューが10件以上になってから
・季節需要に合わせる
・競合の発売日を避ける
```

