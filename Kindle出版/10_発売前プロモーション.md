# Kindle本の発売前プロモーション計画

あなたはKindle本のマーケティング専門家です。

以下の情報をもとに、**発売前から話題を作るプロモーション計画**を立ててください。

【目的】
- 発売初日から売上を最大化
- 事前期待値の醸成
- 初動でランキング入りを狙う

【入力】
- 本のタイトル：[例：月5万円を確実に稼ぐ副業入門]
- 発売予定日：[例：2024年4月1日]
- 活用可能なメディア：[例：Twitter(フォロワー1,000人)、メルマガ(500人)]
- 予算：[例：0円/5,000円/10,000円]
- 著者の強み：[例：副業で実績あり]

【出力形式】

---

**【発売4週間前】認知フェーズ**

```
目標：本の存在を知ってもらう

□ 執筆中の様子をSNSで発信
  投稿例：
  「副業で月5万円稼ぐ方法をまとめた本を執筆中📚
   会社員時代に知りたかったことを全て詰め込んでます。
   発売は4月1日予定。お楽しみに！」

□ 本の内容に関連する有益な投稿を増やす
  テーマ：[関連テーマ1]、[関連テーマ2]

□ メルマガで執筆の裏話を共有
  件名例：「【予告】4月に本を出版します」

□ 表紙デザインの候補を見せてアンケート
  「A、B、Cどの表紙がいいですか？」
```

---

**【発売2週間前】期待醸成フェーズ**

```
目標：「読みたい」と思わせる

□ 表紙を公開して反応を集める
  投稿例：
  「表紙が完成しました！
   【月5万円を確実に稼ぐ副業入門】
   4/1発売です📕」

□ 目次の一部を公開
  「第3章の目次を先行公開👀
   ・STEP1：〇〇を決める
   ・STEP2：△△を準備する
   これだけで月5万円の道筋が見えます」

□ 本の一部を無料で先行公開
  note、ブログ、SNSで「はじめに」を公開

□ 読者の悩みに答える形で本の内容を紹介
  「よく聞かれる質問に、本の中で詳しく答えています」
```

---

**【発売1週間前】カウントダウンフェーズ**

```
目標：発売日を記憶させる

□ カウントダウン投稿（毎日）
  「発売まであと7日！」
  「発売まであと3日！」
  「いよいよ明日発売！」

□ 読者限定特典を発表
  「購入者限定で〇〇をプレゼント🎁」

□ 先行レビュー依頼
  信頼できる人に献本してレビュー依頼

□ 予約ページへの誘導
  「Amazonで予約受付中です！」

□ 発売記念キャンペーン告知
  「発売から3日間は特別価格！」
```

---

**【発売当日】ローンチフェーズ**

```
目標：初日に最大の売上を作る

□ 朝イチで発売告知投稿
  「本日発売！
   【月5万円を確実に稼ぐ副業入門】
   Kindle Unlimitedで無料で読めます📚
   [URL]」

□ メルマガで発売案内を配信
  件名：「【本日発売】〇〇の本が出ました！」

□ ストーリーズ/リールで動画告知
  著者からのメッセージ動画

□ 時間をずらして複数回投稿
  朝・昼・夜で異なる切り口で投稿

□ 読者の反応をリポスト/引用RT
  口コミを増幅させる
```
