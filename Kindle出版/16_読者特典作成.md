# Kindle本の読者特典作成

あなたはKindle出版の特典戦略専門家です。

以下の情報をもとに、**読者の満足度を高め、リスト獲得にもつながる特典**を設計してください。

【目的】
- 読者満足度の向上
- 購入の後押し
- メールリストの獲得
- 口コミ・レビューの促進

【入力】
- 本のタイトル：[例：月5万円を確実に稼ぐ副業入門]

- 著者のリソース：[例：テンプレート作成可、動画なし]
- 目的の優先度：[例：リスト獲得 > 満足度向上]

**推奨特典パッケージ**

```
【パッケージA：シンプル型】
特典1つ + メールアドレス登録

特典例：
「副業スタートチェックリスト（PDF）」
・本書の内容を実践するための20項目
・印刷して使える
・進捗確認に最適

【パッケージB：充実型】
特典3つ + メールアドレス登録

特典例：
① 副業スタートチェックリスト（PDF）
② 収支管理テンプレート（Excel）
③ 著者への質問権（1回）

【パッケージC：プレミアム型】
特典5つ + メールアドレス登録 + コミュニティ

特典例：
① 副業スタートチェックリスト（PDF）
② 収支管理テンプレート（Excel）
③ 本書の要点まとめPDF
④ 著者からのメッセージ動画（5分）
⑤ 読者限定コミュニティ招待
```
