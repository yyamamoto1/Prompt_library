**入稿前チェックリスト**

```
【原稿ファイル】
□ 対応形式で保存（Word/.docx、EPUB、KPF）
□ 目次が正しくリンクしている
□ フォントが埋め込まれている
□ 画像が適切に配置されている
□ ファイルサイズが650MB以下

【表紙ファイル】
□ JPEG or TIFF形式
□ 推奨サイズ：2560 x 1600 px（縦横比1.6:1）
□ 最小サイズ：1000 x 625 px
□ 解像度：300dpi以上
□ RGB カラーモード
□ ファイルサイズ50MB以下
```

**入稿後の確認事項**

```
□ ステータスが「レビュー中」になった
□ 24〜72時間以内に審査完了
□ 販売開始通知メールが届いた
□ 商品ページで正しく表示されている
□ サンプルダウンロードで目次を確認
□ 「なか見！検索」が有効になった

【問題が発生した場合】
・原稿の品質問題 → 該当箇所を修正して再アップロード
・表紙の問題 → 仕様に合わせて修正
・メタデータの問題 → 設定を修正
```

---

**よくあるトラブルと対処法**

```
【原稿関連】
問題：目次がリンクしない
対処：見出しスタイルを正しく適用し直す

問題：改ページがおかしい
対処：ページ区切りを手動で挿入

問題：画像が表示されない
対処：画像をJPEG形式で再挿入

【表紙関連】
問題：画像がぼやける
対処：2560 x 1600 px以上で再作成

問題：アップロードエラー
対処：ファイル形式・サイズを確認

【審査関連】
問題：審査に時間がかかる
対処：72時間経過後にサポートへ連絡
```

---

💡 **入稿のポイント**
- プレビューは必ず全ページ確認
- 審査前に複数端末で表示確認
- 問題があれば修正して再入稿可能
