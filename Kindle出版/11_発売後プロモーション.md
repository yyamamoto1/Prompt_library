# Kindle本の発売後プロモーション戦略

あなたはKindle本の継続的なマーケティング専門家です。

以下の情報をもとに、**長期的に売れ続けるためのプロモーション戦略**を立ててください。

【目的】
- 発売後も継続的に売上を維持
- ロングテールでの収益確保
- レビュー獲得と改善サイクル

【入力】
- 本のタイトル：[例：月5万円を確実に稼ぐ副業入門]
- 発売からの期間：[例：1週間/1ヶ月/3ヶ月]
- 現在の状況：[例：売上10冊、レビュー2件]
- 活用可能なメディア：[例：SNS、ブログ、メルマガ]
- 競合の動き：[例：類書が増えてきた]

【出力形式】

---
📈 **発売後プロモーション戦略**

**フェーズ別戦略**

```
【発売1週間〜1ヶ月】基盤構築期
 ↓
【発売1〜3ヶ月】成長期
 ↓

```

---

**【発売1週間〜1ヶ月】基盤構築期**

```
目標：レビュー10件を集める

□ 購入者へレビュー依頼メール
  「お読みいただきありがとうございます。
   感想をAmazonレビューでいただけると励みになります」

□ SNSでの読者反応を収集・引用
  「読者さんからこんな嬉しい声が！」

□ 読み放題ページ数の推移をチェック
  KDPダッシュボードで毎日確認

□ 初期レビューへの対応
  ★1-2レビュー：真摯に受け止め改善
  ★4-5レビュー：感謝を伝える

□ 関連コンテンツの発信継続
  本の内容を小出しにしてSNSで発信
```

---

**【発売1〜3ヶ月】成長期**

```
目標：カテゴリランキング維持・上昇

□ 期間限定セールの実施
  Kindleカウントダウンディール活用
  期間：3〜5日間
  告知：SNS + メルマガ

□ 関連キーワードでのSEO強化
  商品説明文の最適化
  A+コンテンツの追加（可能な場合）

□ 外部メディアへの露出
  - ブログ記事化
  - note記事化
  - YouTube/ポッドキャストで紹介

□ 読者の声を使った二次プロモーション
  「読者満足度95%！」
  「〇〇さんから推薦の声をいただきました」

□ 内容のアップデート検討
  読者フィードバックを反映した改訂版
```

---

**【発売3ヶ月以降】安定期**

```
目標：継続的な売上の仕組み化

□ 定期的なセール実施（月1回）
  タイミング：月初め or 給料日後

□ 季節イベントに合わせた訴求
  - 新年：「今年こそ副業を始めよう」
  - 新生活：「新社会人におすすめ」
  - ボーナス期：「自己投資に」

□ 関連本の出版
  シリーズ化で相互送客
  - 続編：応用編
  - 関連：〇〇専門編

□ まとめ買い/シリーズ割引
  複数冊購入で割引

□ アフィリエイト活用
  読者にアフィリエイト紹介を依頼
```

---

**レビュー獲得戦略**

```
【直接依頼】
・本の「おわりに」でレビューをお願い
・購入者へのフォローアップメール
・SNSでの声かけ

【間接促進】
・読書メモテンプレートを特典で配布
 →「メモをそのままレビューにどうぞ」
・レビューキャンペーン（倫理的範囲で）

【レビュー対応】
★5-4：感謝のリプライ
★3：改善点の検討
★2-1：真摯に受け止め、可能なら改訂
```

---

**売上低迷時の対処法**

```
状況別アクション：

【レビューが増えない】
→ SNSでの露出を増やす
→ 信頼できる人に献本

【競合に負けている】
→ 価格の見直し
→ 商品説明文の改善
→ 表紙のリニューアル検討

【読み放題ページ数が伸びない】
→ 冒頭部分の改善（離脱防止）
→ 目次の見直し

【ランキングが下がった】
→ 期間限定セール実施
→ 新規チャネルでの告知
```
