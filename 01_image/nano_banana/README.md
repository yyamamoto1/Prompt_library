# Nano Banana Pro プロンプトコレクション

Google の **Nano Banana Pro (Gemini 2.5 Flash Image)** AI画像生成モデル用の包括的なプロンプト集です。

## 📊 コレクション概要

3つの主要GitHubリポジトリから厳選された **302個のプロフェッショナルプロンプト**を収録しています。

| リポジトリ | プロンプト数 | 特徴 |
|-----------|------------|------|
| **ZeroLu** | 70 | フォトリアリズム、クリエイティブ実験、Eコマース |
| **YouMind-OpenLab** | 132 | SNS投稿、インフォグラフィック、ゲームアセット |
| **JimmyLv** | 100 | キャラクターデザイン、スタイル変換、3D効果 |

## 📝 プロンプトの特徴

各プロンプトには以下が含まれます：

✅ **完全なプロンプトテキスト** - そのまま使用可能
✅ **技術仕様** - カメラ設定、照明、解像度などの詳細
✅ **ソースクレジット** - 作成者のTwitterハンドルとリンク
✅ **使用例** - 生成された画像の例（一部）
✅ **カスタマイズ可能なパラメータ** - 柔軟な調整が可能
✅ **多言語対応** - 英語、中国語、日本語、ドイツ語など

## 🚀 使用方法

1. **カテゴリを選択** - 目的に応じたフォルダ（zerolu/youmind/jimmylv）を選択
2. **プロンプトを選択** - 具体的な用途に最も近いプロンプトファイルを開く
3. **カスタマイズ** - 必要に応じてパラメータを調整
4. **生成** - Nano Banana Pro または互換性のあるAIモデルで使用
