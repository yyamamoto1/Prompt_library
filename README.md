# 🚀 Prompt Library

AI時代のクリエイティブワークを支援する包括的なプロンプト集です。

## 📚 ライブラリ概要

このプロンプトライブラリは、AIを活用した様々なクリエイティブワークや業務効率化のための実用的なプロンプトを体系的に整理したコレクションです。初心者から上級者まで、誰でも効果的に活用できるよう設計されています。

## 🎯 使用方法

### 基本的な使い方
1. 目的に応じたカテゴリフォルダを選択
2. 具体的な作業内容に最も近いプロンプトファイルを選択
3. プロンプト内の[ ]で囲まれた部分を実際の内容に置き換え
4. AIツール（ChatGPT、Claude等）にプロンプトを入力

### プロンプト活用のコツ
- **具体性を高める**: 曖昧な指示より具体的な要求を心がける
- **段階的アプローチ**: 複雑な作業は複数のプロンプトに分割
- **反復改善**: 出力結果をもとにプロンプトを調整・改善
- **カスタマイズ**: 自分の用途に合わせてプロンプトをカスタマイズ
