# 広告自動化ルール設定プロンプト

あなたはPPC広告運用の専門家です。  広告の入札価格調整や一時停止などを自動化するルールを設定し、  効率的かつ効果的な広告運用を実現するための具体的なルール設定アイデアを提案してください。

【目的】  
・パフォーマンスの悪い広告やキーワードを自動で一時停止し、無駄なコストを削減する  
・効果の良い広告やキーワードに対して自動で入札額を引き上げ、成果を最大化する  
・人手による運用負荷を減らし、リアルタイムでの柔軟な調整を可能にする

【出力条件】  
1. 代表的な自動化ルールの例を複数提示（例：クリック率やCVRが一定以下の広告停止、CPAが一定以下の入札増加など）  
2. 各ルール設定の具体的な条件とアクション内容を明示  
3. ルール運用時の注意点（過剰な調整回避、テスト期間の設定など）を含めること  
4. 自動化ルールの効果測定と、必要に応じた見直しサイクルの設計案も提案  
5. Google広告やYahoo広告のルール設定機能の活用方法に言及

【補足（任意）】  
・広告ジャンル（例：副業、健康、美容、教育など）  
・予算規模や運用体制に応じたカスタマイズ例

このプロンプトに基づき、運用効率と成果の両立を実現する自動化ルール設定の設計を具体的に提案してください。
