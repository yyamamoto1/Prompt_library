# 広告表示オプション活用プロンプト（Google広告向け）

あなたはGoogle広告運用の専門家です。  サイトリンク表示オプション、コールアウト表示オプション、構造化スニペット表示オプションなど、  複数の広告表示オプションを駆使して、広告の視認性を高め、クリック率（CTR）を最大化する戦略を提案してください。

【目的】
・広告表示領域を拡張し、ユーザーの目を引く情報量を増やす  
・ユーザーの疑問や関心に先回りして回答し、広告の魅力を具体的に伝える  
・広告文だけでは伝えきれない強みや特長を補完的にアピールする  
・クリック率向上とともに品質スコアの改善も狙う

【出力条件】
1. サイトリンク表示オプションの効果的な設定例（例：重要なページへの誘導リンク）と文言案を提示  
2. コールアウト表示オプションで強調すべき特徴やメリット（例：無料相談、返金保証、24時間対応）を具体例付きで示す  
3. 構造化スニペット表示オプションの活用方法と、使うべきカテゴリ（例：サービス内容、ブランド名、地域など）を提案  
4. これらのオプションを組み合わせたときの広告見え方イメージと、それぞれの役割を解説  
5. 効果測定のポイント（CTR変化、品質スコア向上など）と改善サイクルを含めること

【補足（任意）】
・業種や商材ジャンル（例：副業教材、化粧品、学習サービスなど）  
・広告のターゲット（例：初心者、副業希望者、20〜40代女性）

このプロンプトに基づいて、広告表示オプションの最適活用策を具体的に提案してください。
