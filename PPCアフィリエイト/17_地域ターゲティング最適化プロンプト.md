# 地域ターゲティング最適化プロンプト

あなたはPPC広告運用の専門家です。  特定の地域（都道府県、市区町村など）にターゲットを絞り、  その地域に住む、または関心を持つ「高精度なターゲット（целевой）」に対して効率的に広告配信を行うための  地域ターゲティング設定と広告クリエイティブの最適化戦略を提案してください。

【目的】
・地域ごとの特性やニーズに合わせた訴求でCTRとCVRを最大化する  
・無駄な地域配信を減らし、広告費の無駄遣いを防止する  
・地域別の反応データを活用して、広告文・画像・オファーの微調整を行う

【出力条件】
1. 地域ターゲティング設定の具体的な方法（Google広告・Yahoo広告の設定例含む）を説明  
2. 地域特性に応じた広告クリエイティブ調整のポイント（例：地域限定キャンペーン、地元の言葉・文化を活かす表現）を提案  
3. 地域別パフォーマンスの分析方法と、改善に活かすためのKPI設計を示す  
4. 複数地域を扱う場合のセグメント分け・予算配分の考え方も含めること  
5. 地域ターゲティングで効果を最大化するための運用サイクルを提案

【補足（任意）】
・商品ジャンル（例：飲食店、教育サービス、美容クリニックなど）  
・ターゲット属性（例：30〜50代女性、子育て世代、ビジネスマンなど）

このプロンプトに従い、地域ごとのターゲットに刺さる広告配信設計とクリエイティブ最適化策を具体的に提案してください。
