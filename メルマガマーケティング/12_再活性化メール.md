# 休眠読者再活性化メール

あなたはメールマーケティングの再活性化専門家です。

以下の情報をもとに、**休眠読者を復活させるメールシリーズ**を作成してください。

【目的】
- 休眠読者のエンゲージメント回復
- リストの品質維持
- 無関心な読者の整理

【入力】
- 休眠の定義：[例：90日以上未開封]
- 休眠読者数：[例：1000人]
- リストの特性：[例：副業に興味がある人]

【出力形式】

---
🔄 **再活性化メールシリーズ（3通）**

**第1通：久しぶりのご挨拶**

件名：「[名前]さん、お久しぶりです」

```
[名前]さん

お久しぶりです。
[メルマガ名]の[送信者名]です。

最近、[名前]さんにメールを
お届けしていましたが、
お忙しかったでしょうか？

実は最近、[テーマ]について
とても役立つ情報を
お届けしています。

もし引き続き受け取っていただけるなら、
こちらをクリックしてください。

▼ 引き続き受け取る
[確認リンク]

もちろん、不要でしたら
下記から配信停止できます。

[配信停止リンク]

[署名]
```

件名：「見逃していませんか？」

```

お読みいただけましたか？

```

**第3通：最終確認（7日後）**

件名：「最後の確認です」

```
[名前]さん

これが最後のご確認メールです。

今後もメルマガを受け取りたい方は、
下のボタンをクリックしてください。

▼ 受け取りを継続する
[確認リンク]

配信リストから削除させていただきます。

ありがとうございました。

[署名]
```

📋 **再活性化後のフォロー**
- 継続した読者 → 通常配信へ
- 反応なし → リストから削除（クリーニング）
