# メルマガセグメント配信戦略

あなたはメールマーケティングのセグメンテーション専門家です。

以下の情報をもとに、**効果的なセグメント配信戦略**を設計してください。

【目的】
- 読者に最適化されたコンテンツ配信
- 開封率・クリック率の向上
- 解除率の低下

【入力】
- リスト総数：[例：5000人]
- 取得している情報：[例：名前、メール、登録日、開封履歴]
- 商品ラインナップ：[例：初級/中級/上級コース]
- 現在の配信方法：[例：全員に同じ内容]

【出力形式】

---
🎯 **セグメント配信戦略**

**セグメント分類**

① 登録経路別
- ブログ経由：[内容の方向性]
- SNS経由：[内容の方向性]
- 広告経由：[内容の方向性]

② 行動履歴別
- 高エンゲージメント（開封率50%以上）
  → VIP扱い、先行案内
- 中エンゲージメント（開封率20-50%）
  → 通常配信
- 低エンゲージメント（開封率20%以下）
  → 再活性化キャンペーン

③ 購入履歴別
- 未購入者
  → 教育コンテンツ、低価格商品
- 購入者（1回）
  → 関連商品、アップセル
- リピーター（2回以上）
  → VIP特典、高額商品

④ 登録期間別
- 新規（30日以内）
  → ウェルカムシリーズ
- 中期（31-90日）
  → 通常配信
- 長期（91日以上）
  → ロイヤリティ施策

**セグメント別コンテンツ設計**

| セグメント | 配信内容 | 頻度 |
