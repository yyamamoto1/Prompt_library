# アフィリエイトメール作成

あなたはアフィリエイトメールマーケティングの専門家です。

以下の情報をもとに、**読者の信頼を損なわずに成約率を高めるアフィリエイトメール**を作成してください。

【目的】
- アフィリエイト商品の自然な紹介
- 読者との信頼関係維持
- 成約率の向上

【入力】
- アフィリエイト商品：[例：プログラミングスクール]
- 報酬単価：[例：10,000円/成約]
- 読者層：[例：プログラミング学習に興味がある会社員]
- 紹介の理由：[例：自分も受講して良かった]

【出力形式】

---
📧 **アフィリエイトメール本文**

**パターン①：体験談ベース**

```
件名：私が[目標]を達成できた理由

[名前]さん

実は今日は、私が[成果]を達成できた
きっかけについてお話しさせてください。

[自分のストーリー：過去の悩み]

そんな時に出会ったのが
[商品名]でした。

正直、最初は半信半疑でしたが...

[実際に使った感想・結果]

もし[名前]さんも同じような悩みを
お持ちでしたら、チェックしてみてください。

▼ [商品名]の詳細はこちら
[アフィリエイトリンク]

※これは私の個人的な体験であり、
 効果を保証するものではありません。

[署名]
```

```
よく質問をいただくので、

【イマイチな点】

・[デメリット2]

【こんな人におすすめ】
・[ターゲット1]
・[ターゲット2]

▼ 詳細はこちら
[アフィリエイトリンク]

[署名]
```

**パターン③：価値提供+さりげなく紹介**

```
件名：[役立つ情報のタイトル]

[名前]さん

今日は[テーマ]について
お伝えします。

[価値ある情報を7-8割]

ちなみに、[テーマ]に関連して
最近私が使っているのが[商品名]です。

もし興味があれば参考にしてみてください。
[アフィリエイトリンク]

[署名]
```

---

⚠️ **アフィリエイトメールの注意点**
- 広告である旨を明記
- 嘘や誇大表現は禁止
- 読者のためになる商品のみ紹介
- 頻度は月1-2回程度に抑える
