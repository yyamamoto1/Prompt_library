# 価格提示戦略と正当化

あなたは価格戦略とセールスの専門家です。

以下の情報をもとに、**価格を適切に提示し、購入を促進する戦略**を設計してください。

【目的】
- 価格の正当性を伝える
- 価値 > 価格と感じさせる
- 購入の心理的ハードルを下げる

【入力】
- 商品/サービス：[例：Webマーケティング講座]
- 価格：[例：198,000円（税込）]
- 競合価格帯：[例：20-50万円]
- 提供価値：[例：6ヶ月のサポート、添削、質問無制限]
- 特典（あれば）：[例：テンプレート集、個別コンサル]

【出力形式】

---
💰 **価格提示戦略**

**価格設定の心理学**

```
【価格提示の原則】

① 価値を先に伝える
   価格を見せる前に十分な価値を伝える

② アンカリング効果
   高い比較対象を先に見せる

③ 分割表示
   大きな金額は分割で見せる

④ リスクリバーサル
   返金保証で安心感を与える
```

**特典・ボーナスの見せ方**

```
【特典提示の構成】

■ メイン商品
[商品名]：[価格]

■ さらに今回だけ...

特典1：[特典名]（[X]万円相当）
→ [特典の説明・ベネフィット]

特典2：[特典名]（[X]万円相当）
→ [特典の説明・ベネフィット]

特典3：[特典名]（[X]万円相当）
→ [特典の説明・ベネフィット]

---

【特典の効果的な見せ方】
・「○万円相当」で価値を明示
・なぜ今回だけなのか理由を説明
・本当に価値のある特典を用意
・多すぎない（3-5個程度）
```

---

**保証・リスクリバーサル**

```
【返金保証のパターン】

■ 無条件返金保証

■ 条件付き返金保証
「カリキュラムを全て実践して
 成果が出なければ全額返金」

■ 成果保証
「[成果]を達成するまでサポート延長」
```

**緊急性・限定性の演出**

```
【時間的限定】
・「本日23:59まで」
・「[日付]までの限定価格」
・「早期割引は今週末まで」

【数量限定】
・「先着30名様限り」
・「残り5名」
・「今月の募集枠：10名」

【条件限定】
・「初回購入の方限定」
・「メルマガ読者限定」
・「このページからの申込み限定」

【注意点】
・嘘の緊急性は信頼を失う
・本当の理由がある限定にする
```

---

**価格セクションの構成例**

```
【推奨構成】

1. 価値のまとめ
   「ここまでの内容をまとめると...」

2. 総額価値の提示
   「通常○万円相当の内容です」

3. 特別価格の提示
   「今回は特別に...」

4. 分割払いオプション
   「月々○円から始められます」

5. 特典の追加
   「さらに今回だけ...」

6. 保証の提示
   「安心の返金保証付き」

7. 限定性
   「ただし、○名様限り」

8. CTA
   「今すぐ始める」
```

---

💡 **価格提示のポイント**
- 価格の前に十分な価値を伝える
- 比較対象で安く感じさせる
- 保証でリスクを取り除く
- 今行動する理由を明確に
