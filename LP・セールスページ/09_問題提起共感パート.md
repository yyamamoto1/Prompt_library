# 問題提起・共感パート作成

あなたはセールスライティングの専門家です。

以下の情報をもとに、**読者の悩みに共感し、解決策への期待を高める問題提起パート**を作成してください。

【目的】
- 読者の悩みを言語化
- 「わかってくれている」と感じさせる
- 解決策（商品）への期待を高める

【入力】
- ターゲット：[例：副業で稼ぎたい30代会社員]
- 主な悩み：[例：時間がない、何をすればいいかわからない]
- 解決する商品：[例：副業自動化システム]
- ターゲットの状況：[例：本業が忙しく、残業も多い]

【出力形式】

   「でも、解決方法があります」

**悩みの深掘り・共感**

```
【深掘りの構造】

表面的な悩み → 本質的な悩み → 感情

---

【例文】

「時間がない」

これは表面的な悩みです。

本当の悩みは...

・家族との時間を犠牲にしている罪悪感
・いつまでこの生活が続くのかという不安
・自分の人生を自分でコントロールできない苛立ち

そして心の奥では...

「本当はもっと自由に生きたい」
「会社に縛られない人生を歩みたい」
「家族とゆっくり過ごす時間が欲しい」

こんな思いを抱えていませんか？

わかります。
私も同じでした。

---

【共感フレーズ集】

・「その気持ち、よくわかります」
・「私も以前は同じ悩みを抱えていました」
・「多くの方が同じ壁にぶつかっています」
・「あなたは一人ではありません」
・「その悩み、解決できます」
```

---

**原因の提示**

```
【なぜ悩みが解決しないのか】

「なぜ、うまくいかないのか？」

それは、あなたのせいではありません。

問題は[根本原因]にあります。

---

【原因提示の例】

■ 情報過多型
「ネットには情報が溢れすぎています。
 何が正しいのかわからず、
 行動できないのは当然です」

■ 方法論の問題型
「従来の方法では、
 時間がない人には不向きなのです。
 根本的に仕組みを変える必要があります」

■ 構造的問題型
「そもそも、労働収入には限界があります。
 時間を切り売りする働き方では、
 自由を手に入れることはできません」

---

【原因→解決への橋渡し】

「でも、安心してください。

[原因]を解決する方法があります。

それが[商品名/解決策]です」
```

---

**共感パート全体の例文**

```
━━━━━━━━━━━━━━━━━━━━
【完成例】
━━━━━━━━━━━━━━━━━━━━

■ ヘッドライン
「副業したいけど時間がない...」
そんなあなたへ

■ 問題提起
毎日、仕事に追われていませんか？

・朝から晩まで働いて、帰宅は夜10時
・副業したいけど、まとまった時間がない
・休日は疲れて寝てしまい、何もできない
・このままでは一生、会社員のまま...

■ 共感
その気持ち、よくわかります。

私も3年前まで、同じ状況でした。
毎日満員電車に揺られ、残業の日々。
副業を始めたいと思いながらも、
何も行動できない自分に
焦りと苛立ちを感じていました。

■ 深掘り
本当の悩みは「時間がないこと」ではなく、
「このまま変われない自分」への不安
ではないでしょうか。

・いつまでこの生活が続くのか
・定年まで働き続けられるのか
・もっと自由に生きられる方法はないのか

■ 原因
なぜ、多くの人が副業で挫折するのか。

それは「時間を切り売りする副業」を
選んでしまうからです。

本業で忙しいのに、
さらに時間を使う副業は続きません。

■ 希望
でも、安心してください。

「時間を使わずに収入を得る」
仕組みを作ることができます。

私はこの方法で、
本業を続けながら月[X]万円の
副収入を得られるようになりました。

その方法を、今日お伝えします。
```

---

💡 **共感パート作成のポイント**
- ターゲットの言葉で悩みを表現
- 表面的な悩みから本質に迫る
- 「わかってくれている」と感じさせる
- 希望を持たせて次のセクションへ
