# 実績・権威性セクション作成

あなたは信頼構築とブランディングの専門家です。

以下の情報をもとに、**信頼性と権威性を高めるセクション**を設計してください。

【目的】
- 信頼性の構築
- 購入の安心感を与える
- 競合との差別化

【入力】
- 商品/サービス：[例：ビジネスコンサルティング]
- 実績：[例：累計500社支援、売上向上率98%]
- 資格・経歴：[例：MBA取得、元上場企業役員]
- メディア掲載：[例：日経新聞、ダイヤモンド社]
- 受賞歴：[例：ベストコンサルタント賞]

【出力形式】

---
🏆 **実績・権威性セクション設計**

**権威性構築の要素**

```
【信頼性の7つの柱】

① 実績（数字）
② 資格・認定
③ 経歴・経験
④ メディア掲載
⑤ 受賞・表彰
⑥ 推薦・推奨
⑦ 顧客の声
```

```

・「総売上[X]億円達成を支援」

・「[X]万円のコスト削減」

【推薦者の選び方】

・業界の権威者

・メディア関係者

・同業の専門家
・取引先の代表
・長年の顧客

```

---

**セクション構成例**

```
【権威性セクション全体構成】

■ セクションタイトル
「選ばれる理由」
「なぜ〇〇が選ばれるのか」

■ 実績数字（横並び）
[数字1] [数字2] [数字3]

■ プロフィール
写真 + 経歴 + 資格

■ メディア掲載
ロゴ一覧

■ 受賞・認定
バッジ一覧

■ 推薦コメント
権威者からの推薦
```

---

💡 **権威性構築のポイント**
- 具体的な数字を使う
- 第三者の評価を活用
- 視覚的に見やすく整理
- 嘘や誇張は厳禁
