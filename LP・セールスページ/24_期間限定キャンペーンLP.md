# 期間限定キャンペーンLP

あなたはプロモーションとキャンペーン設計の専門家です。

以下の情報をもとに、**期間限定キャンペーンで成果を最大化するLP**を設計してください。

【目的】
- 期間内の売上最大化
- 緊急性による購買促進
- 新規顧客の獲得

【入力】
- 商品/サービス：[例：オンライン講座]
- 通常価格：[例：98,000円]
- キャンペーン価格：[例：49,800円]
- キャンペーン期間：[例：12/20-12/25の5日間]
- キャンペーン理由：[例：クリスマス特別企画]

【出力形式】

---
🎉 **キャンペーンLP設計**

**キャンペーンLPの特徴**

```
【通常LPとの違い】

・緊急性が明確
・期間/数量が限定
・特別オファーがある
・今行動する理由が明確

【成功するキャンペーンの要素】

① 明確な理由
② 魅力的なオファー
③ 緊急性・限定性
④ シンプルな構成
⑤ 強力なCTA
```

**キャンペーンスケジュール**

```
【プロモーション計画】

■ 事前告知（3-7日前）
・ティーザー配信
・期待感の醸成

■ キャンペーン開始
・全力告知
・初日に勢いをつける

■ 中盤リマインド
・途中経過報告
・残り枠の告知

■ 最終日（ラストスパート）
・「本日最終日」告知
・カウントダウン
・最終リマインド

■ 終了
・終了の告知
・購入者へのフォロー
```
