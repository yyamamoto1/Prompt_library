# お客様の声セクション作成

あなたはソーシャルプルーフ（社会的証明）の専門家です。

以下の情報をもとに、**信頼性を高めるお客様の声セクション**を設計してください。

【目的】
- 信頼性の構築
- 購入への不安払拭
- 共感と期待の創出

【入力】
- 商品/サービス：[例：オンライン英会話]
- ターゲット：[例：ビジネスで英語が必要な30代会社員]
- 収集済みの声：[例：3件/これから収集]
- 強調したい成果：[例：3ヶ月でTOEIC200点UP]

【出力形式】

---
⭐ **お客様の声セクション設計**

**効果的な testimonial の要素**

```
【信頼性を高める要素】

✅ 実名・顔写真
✅ 具体的な属性（職業、年齢など）
✅ Before/Afterの変化
✅ 具体的な数字
✅ ストーリー性
✅ ターゲットと似た属性
```

---

**お客様の声のテンプレート**

```
【パターン1：変化ストーリー型】
━━━━━━━━━━━━━━━━━━━━

■ 構成
1. Before（以前の状態・悩み）
2. きっかけ（なぜ始めたか）
3. プロセス（どう取り組んだか）
4. After（どう変わったか）
5. メッセージ（検討中の人へ）

■ 例文
「英語が苦手で、会議で発言できずにいました。
このサービスを3ヶ月続けた結果、
今では海外のクライアントと
堂々とミーティングができています。
迷っているなら、まず試してみてください」

[名前] [年齢][職業]

---

【パターン2：成果フォーカス型】
━━━━━━━━━━━━━━━━━━━━

■ 構成
1. 具体的な成果（数字）
2. どのくらいの期間で
3. 何が良かったか
4. 推薦の言葉

■ 例文
「3ヶ月でTOEIC650→850点に！
毎日30分の学習だけでここまで伸びました。
カリキュラムが実践的で、
実際の仕事にすぐ活かせたのが良かったです」

[名前] [成果の数字]

---

【パターン3：不安払拭型】
━━━━━━━━━━━━━━━━━━━━

■ 構成
1. 最初の不安・懸念
2. 実際にやってみて
3. 不安は不要だった理由
4. 今の感想

■ 例文
「正直、オンラインで本当に上達するのか
不安でした。でも、マンツーマンで
自分のペースで学べるので、
むしろ対面より効率的でした」

[名前] [属性]
```

**お客様の声の収集方法**

```
【収集タイミング】
□ 成果が出た時
□ サービス利用後1ヶ月
□ コース修了時
□ 更新・リピート時

【収集方法】
□ アンケートフォーム
□ インタビュー
□ メールでの依頼
□ ビデオ収録

【質問テンプレート】

Q1. サービスを利用する前、
    どんな悩み・課題がありましたか？

Q2. 何がきっかけで利用を決めましたか？

Q3. 利用してみて、どんな変化がありましたか？
    （具体的な成果があれば教えてください）

Q4. 特に良かった点は何ですか？

Q5. 同じ悩みを持つ方に一言お願いします。

Q6. 写真・実名の掲載許可をいただけますか？
```

---

**信頼性を高める工夫**

```
【必須要素】
✅ 実名（苗字 + 名前の頭文字でも可）
✅ 顔写真（イラストより写真）
✅ 属性情報（職業、年齢など）
✅ 具体的な成果・数字

【あると良い要素】
○ 手書きの文字
○ 動画での証言
○ SNSの埋め込み
○ 会社名・役職（許可があれば）

【避けるべきこと】
✗ 匿名すぎる（「30代女性」のみ）
✗ 抽象的すぎる感想
✗ 明らかに作られた感のある文章
✗ 似たような声ばかり
```

---

💡 **お客様の声活用のポイント**
- ターゲットと似た属性の人を優先
- 具体的な数字・成果を入れる
- 不安を払拭する声も入れる
- 定期的に新しい声を追加
