# 緊急性・限定性の演出

あなたはセールスプロモーションの専門家です。

以下の情報をもとに、**今すぐ行動を促す緊急性・限定性の演出**を設計してください。

【目的】
- 即座の行動を促す
- 先延ばしを防ぐ
- 購買意欲を高める

【入力】
- 商品/サービス：[例：オンラインセミナー]
- 通常価格：[例：50,000円]
- キャンペーン：[例：早期割引]
- 実施可能な限定：[例：人数制限あり]
- 期間：[例：1週間限定]

【出力形式】

---
⏰ **緊急性・限定性設計**

**心理的原理**

```
【なぜ緊急性が効くのか】

① 損失回避
   人は「得る」より「失う」を恐れる
   「今だけ」を逃したくない心理

② 希少性の原理
   手に入りにくいものほど価値が高く感じる

③ 社会的証明
   「みんなが欲しがっている」と感じさせる

④ 決断の後押し
   迷っている人の背中を押す
```

**限定性の根拠と説明**

```
【なぜ限定なのか理由を説明】

理由がないと信頼されない
嘘の限定は信頼を失う

---

【理由の例】

■ キャパシティの制限
「一人ひとりにしっかりサポートするため、
 月間の受入れ人数を10名に制限しています」

■ リソースの制限
「講師のスケジュールの都合上、
 今回の募集は5名様限りとなります」

■ テスト価格
「正式リリース前のモニター価格のため、
 この価格でのご提供は今回限りです」

■ 周年・記念
「3周年を記念した特別価格です。
 通常価格に戻る前にお申し込みください」

■ 在庫・仕入れ
「次回入荷が未定のため、
 現在の在庫限りとなります」
```

**注意点とベストプラクティス**

```
【やってはいけないこと】

❌ 嘘の緊急性
   「毎日今日限り」は信頼を失う

❌ 根拠のない限定
   なぜ限定か説明がないと疑われる

❌ 過度な煽り
   「今買わないと一生後悔」など

❌ 常にセール
   常に割引だと価値が下がる

---

【ベストプラクティス】

✅ 本当の理由がある限定にする
✅ 限定の根拠を説明する
✅ 約束したら守る（延長しない）
✅ 適度な頻度で使う
✅ 押し売りではなく情報提供として
```

**緊急性セクション構成例**

```
【推奨構成】

1. 注意喚起
   「重要なお知らせ」

2. 限定の内容
   「今回は〇名様限定」

3. 理由の説明
   「なぜ限定なのか」

4. 残り状況
   「現在残り〇名」

5. 期限
   「〇月〇日まで」

6. 行動喚起
   「今すぐお申し込みください」

7. CTA
   [申し込みボタン]
```

---

💡 **緊急性演出のポイント**
- 嘘のない本当の限定を作る
- 理由を明確に説明する
- 視覚的にわかりやすく
- 煽りすぎない適度なトーン
