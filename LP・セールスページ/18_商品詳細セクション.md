# 商品詳細セクション作成

あなたはプロダクトマーケティングの専門家です。

以下の情報をもとに、**商品の内容を魅力的に伝える詳細セクション**を設計してください。

【目的】
- 商品内容の明確な説明
- 価値の可視化
- 購入判断の材料を提供

【入力】
- 商品/サービス：[例：Webデザイン講座]
- 主な内容：[例：動画講座、添削、コミュニティ]
- 提供形式：[例：オンライン/対面]
- 期間：[例：6ヶ月]
- 含まれるもの：[例：動画50本、テンプレート、サポート]

【出力形式】

---
📦 **商品詳細セクション設計**

**商品詳細の役割**

```

・含まれる内容の詳細
・各内容の価値・ベネフィット
・提供形式・アクセス方法
・サポート内容
```

**価値の見せ方**

```
【価値を最大化する表現】

■ 数量を強調
・「全50本の動画」
・「100種類のテンプレート」
・「30時間分のコンテンツ」

■ 時間換算
・「通常習得に2年かかる内容を
   6ヶ月に凝縮」
・「1日30分で完了できる設計」

■ 金額換算
・「通常〇万円相当の内容」
・「1本あたり〇円の価値」

■ 手間の省略
・「これだけで完結」
・「他に買い足す必要なし」
・「すぐに使える」
```

**商品詳細セクション構成例**

```
【推奨構成】

■ セクションタイトル
「[商品名]の内容」
「あなたが手に入れるもの」

■ 全体概要（1-2行）
商品の全体像を簡潔に

■ 詳細コンテンツ（3-5項目）
含まれるものを詳しく説明

■ まとめ
「これら全てが含まれています」

■ CTA（任意）
「今すぐ手に入れる」
```

💡 **商品詳細作成のポイント**
- 何が含まれるか明確に
- 各要素のベネフィットも記載
- 視覚的にわかりやすく
- ボリューム感を演出
