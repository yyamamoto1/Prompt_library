# 選ばれる理由セクション作成

あなたは競合分析と差別化戦略の専門家です。

以下の情報をもとに、**競合と差別化し選ばれる理由を伝えるセクション**を設計してください。

【目的】
- 競合との違いを明確化
- 選ぶべき理由を提示
- 比較検討中の人の決断を後押し

【入力】
- 商品/サービス：[例：プログラミングスクール]
- 競合：[例：他のスクール、独学、書籍]
- 自社の強み：[例：転職保証、現役エンジニア講師]
- 弱み（あれば）：[例：価格が高め]
- ターゲット：[例：未経験から転職したい人]

【出力形式】

**差別化の考え方**

```
【差別化の3つの軸】

① 機能・スペック
   ・他にない機能
   ・より優れた性能
   ・より多い内容

② 感情・体験
   ・安心感
   ・楽しさ
   ・達成感

③ 価値観・理念
   ・共感できる思い
   ・社会的意義
   ・ブランドストーリー

【差別化の注意点】

・競合を直接批判しない
・自社の強みに焦点を当てる
・証拠・根拠を示す
```

**強みの言語化テンプレート**

```
【強み表現のフレームワーク】

■ [強みのキーワード]

【What】何が違うか
[具体的な違いを説明]

【Why】なぜそれが重要か
[顧客にとっての価値]

【How】どう実現しているか
[根拠・方法]

【Proof】証拠
[実績・データ]

---

【例：転職保証】

■ 転職保証

【What】
転職できなければ全額返金します。

【Why】
「本当に転職できるのか」という
不安を完全に解消。
リスクゼロで挑戦できます。

【How】
カリキュラム修了者の転職成功率98%。
万が一の場合は全額返金の体制を整えています。

【Proof】
これまで1,000名以上が転職成功。
返金実績はわずか2%です。
```

**競合比較の見せ方**

```
【効果的な比較表現】

■ やるべきこと
✅ 自社の強みを明確に
✅ 客観的な事実で比較
✅ 顧客目線でのメリット
✅ 証拠・データを提示

■ 避けるべきこと
❌ 競合の名指し批判
❌ 根拠のない優位性主張
❌ 誇大表現
❌ 競合の虚偽情報

---

【比較の表現例】

良い例：
「当社は現役エンジニアが講師を務めます」
（自社の強みを述べる）

悪い例：
「A社の講師は素人です」
（競合を批判する）

---

【間接的な比較】

「他のスクールでよくある
 『動画を見るだけ』ではありません」

「一般的なスクールと違い、
 実践重視のカリキュラムです」

→ 名指しせず、一般論として比較
```

**セクション構成例**

```

■ セクションタイトル
「[商品名]が選ばれる理由」
「なぜ[商品名]なのか？」
「他との違い」

「多くのお客様に選ばれている
 理由をご紹介します」

■ 理由リスト（3-5個）
それぞれ見出し + 説明 + 証拠

■ まとめ
「だから、[商品名]は
 〇〇な方に選ばれています」

■ CTA（任意）
```

**よくある強みの言い換え例**

```
【陳腐な表現 → 差別化表現】

「サポートが充実」
→「24時間以内に必ず返信。
  質問回数無制限」

「高品質」
→「〇〇認定。業界基準の
  2倍の検査工程」

「実績豊富」
→「10年で5,000件の導入実績。
  継続率98%」

「お客様第一」
→「全額返金保証。
  満足いただけなければ返金」
```

---

💡 **選ばれる理由セクションのポイント**
- 顧客目線での価値を伝える
- 証拠・データで裏付ける
- 競合批判は避ける
- 具体的かつ明確に表現
