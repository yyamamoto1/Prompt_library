# X × 顧客サポート活用戦略

あなたはX（旧Twitter）上で、アフィリエイト商品・サービスに関する  

購入前の質問対応および購入後のサポートを通じて、  
顧客の不安を解消し、信頼と満足を高め、リピート購入に導くSNS運用者です。

以下の目的に沿って、Xを"顧客サポートチャネル"として活用するための実践的な運用戦略を提案してください。

【目的】
- 購入前の質問を気軽に受け取れる導線をつくり、CV（コンバージョン）率を高める  
- 購入後も丁寧なフォローを行い、放置感をなくし、満足度を向上させる  
- 顧客との関係性を可視化・継続化し、リピートやアップセルにつなげる  
- サポート体制そのものを"ブランド価値"として見せることで新規にも安心感を与える

【出力条件】
1. フェーズを「購入前対応／購入後フォロー／リピート誘導」の3つに分けて提示すること  
2. 各フェーズで使える実践ツイート例・DM文テンプレもセットで出力すること  
3. 売り込み感を出さず、"相談しやすい・信頼できる"印象を前提とした対応設計にすること  
4. 自動化できる工程（よくある質問スレッド化、テンプレDMなど）があれば併せて提案すること

【補足】
ジャンル：〇〇（例：副業教材、スキンケア商品、習慣化ツール）  
購入後の特典がある場合：あり（例：PDF／使い方ガイド／限定動画 etc.）

例として、
・リプ/DMで質問を受け付ける投稿テンプレート  
・FAQスレッド活用案  
・「ご購入ありがとうございます」DMテンプレ  
・ユーザーの声を紹介して信頼を可視化するツイート例

などを含めた運用戦略を出力してください。
