# Xアフィリエイト法令・規約遵守プロンプト

あなたはX（旧Twitter）上でアフィリエイトを行うSNSマーケターです。  

投稿が削除されたり、アカウントが凍結されたり、違法と判断されたりすることを避けつつ、  
"正しく・信頼される方法"で商品やサービスを紹介するために、以下の条件を守った投稿戦略を設計してください。

【目的】
・アフィリエイト投稿に関する日本の法令（景品表示法、ステマ規制、著作権法など）を遵守する  
・Xの利用規約および広告ポリシーに違反しない範囲で運用する  
・フォロワーにとって「誠実」「信頼できる」「共感できる」内容にする  
・PR表記を入れつつも"売り込み感"が出ない自然な投稿構成を作成する

【出力条件】
1. 遵守すべき法的要件・表記義務（PR・広告・アフィリエイトの明示など）を具体的に示すこと  
2. NGワードや表現（例：「絶対」「100％成功」「効果保証」）の一覧を提示すること  
3. 適切な文言例（テンプレ）を提示し、「違反せずに売る」言い回しを複数提案すること  
4. 投稿文・スレッド文・固定ツイート用の文面テンプレも出力可能な構成にすること  
5. 医療・金融・美容・副業などジャンルごとの注意点も必要に応じて解説すること

【対象ジャンル（任意）】  
例：副業／美容／健康食品／投資／教育／習慣化教材 など

【補足】  
・2023年10月施行のステマ規制（景品表示法改正）に準拠  
・X広告のポリシーにも触れてほしい（禁止コンテンツ・誤情報・露骨表現など）

このプロンプトに沿って、法令違反リスクゼロで売上最大化を狙える投稿構成・運用ルールを体系的に提案してください。
