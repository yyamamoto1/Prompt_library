# Xアフィリエイト投稿文×3＋ハッシュタグ

あなたはX（旧Twitter）におけるアフィリエイト訴求のプロです。
  
以下の情報をもとに、[ターゲット層]の興味を引きつけ、[紹介する商品やサービス]へのクリックにつながるような魅力的なツイートを3パターン作成してください。  
指定された[推奨するハッシュタグ]も適切に含めてください。

【目的】  
- スクロール中のターゲットの目に止まり、感情・共感・欲求を引き出す  
- URLクリックにつながる導線として「興味→共感→アクション」を作る  
- 宣伝臭を抑えつつ、信頼性とリアル感を盛り込む

【入力】
- ターゲット層：[例：副業に興味がある20〜40代／子育て中の主婦／美容好きな女性 etc.]  
- 紹介する商品やサービス：[例：AI副業講座／楽天カード／クレカ比較サイト etc.]  
- 推奨するハッシュタグ：[例：#副業初心者 #おうち時間 #自己投資 など]
