# ターゲット顧客の深掘り × Instagram投稿最適化

## プロンプト概要
理想的なターゲット顧客を詳細分析し、最も共感される投稿内容と表現方法を設計するプロンプト。

## プロンプト内容

```
あなたはInstagramマーケティングのプロフェッショナルです。  

以下の[商品ジャンル]に興味を持つ理想的なターゲット顧客について、以下の観点で詳細に分析し、  
その顧客が最も共感し、保存・シェアしたくなるような**投稿内容と表現方法（トーン・構図・コピー）**を提案してください。

【目的】  
- 「商品に興味はあるが、まだ動いていない」潜在層を動かす  
- ターゲットの"悩み・欲求・生活感"にジャストで刺さる表現を設計する  
- 単なる宣伝ではなく"共感・納得・信頼"を生む投稿に最適化する

【入力】  
- 商品ジャンル：[例：美容液／時短副業講座／オーガニックおやつ／宅トレグッズ etc.]

【出力形式】

---

### 🧍‍♀️ 理想の顧客像（ペルソナ）

- **年齢／性別：**  
- **職業・収入：**  
- **趣味・関心：**  
- **ライフスタイル：**  
- **抱える悩み：**  
- **SNSの使い方：**（見るだけ／保存メイン／ストーリーズ重視 etc.）

---

### 📱 投稿で刺さる内容テーマ（3つ）

1. **共感投稿型：**  
→ 読者の悩みを代弁し、「私も同じ」と思わせる  
→ 例：「スキンケア迷子だった私がやめた3つのこと」

2. **ストーリー型：**  
→ Before / After、使い始めてからの変化、日常に溶け込む流れ  
→ 例：「正直、面倒だった。でも1週間続けたら…」

3. **ノウハウ・裏技型：**  
→ 商品の活用テクニック・他人に話したくなる知識  
→ 例：「◯◯に1滴混ぜるだけで浸透力が倍になる裏ワザ」

---

### ✍️ 表現方法の提案

- **トーン：** やさしい・共感的・日常語＋一部だけ砕けた本音（例：「マジで救われた」）  
- **ビジュアル構成：** Before/After比較／テロップありの使用風景／手元や生活感ある実写  
- **CTA（行動喚起）：**  
　・「保存してあとで見返して」  
　・「これ、知らなかった人は多分損してる」  
　・「同じ悩みの人、他にいる？」

---

🧠このプロンプトを使えば、**ターゲットが"まるで自分のこと"と感じる精度の高い投稿設計**が可能になります。

もし実行する場合は、以下を入力してください：

- 商品ジャンル（できれば一言特徴も）
  
すぐにペルソナ設計＋投稿構成を出力します。準備OKですか？
```

## 使用方法
1. 商品ジャンルと特徴を明確にする
2. プロンプトを実行してペルソナ分析を取得
3. 3つの投稿テーマから自分に適したものを選択
4. 表現方法の提案を参考に投稿を作成

## 活用場面
- ターゲット設定の精密化
- 共感度の高い投稿作成
- ブランドメッセージの最適化
- 潜在顧客の顕在化
