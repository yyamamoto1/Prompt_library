# Instagram競合アカウント分析 → 戦略提案

## プロンプト概要
競合Instagramアカウントを分析し、自社アカウントの差別化戦略を提案するプロンプト。

## プロンプト内容

```
あなたはInstagramアフィリエイト戦略に長けたSNSマーケターです。  

以下の[競合となるインスタアフィリエイトアカウント]を分析し、  
1. 投稿内容の傾向  
2. エンゲージメント獲得の工夫  
3. プロモーション・CV導線の作り方  
の3つの観点から、自社アカウントに取り入れるべき要素と、差別化すべき点を提案してください。

【目的】  
- 競合の成功要因を客観的に抽出し、ベンチマークとする  
- "似たような発信者"で終わらず、明確な違い・優位性を構築する  
- フォロワー→ファン→購入の流れを最短にする戦略を立てる

【入力】  
- 競合アカウントのアカウント名または投稿URL（複数可）  
- 自社アカウントのジャンルまたは方針（任意）

【出力形式】
---

### 🧩 競合アカウント分析（例：@example_affi）

#### ① 投稿内容の傾向  
- 内容タイプ：HOWTO中心／レビュー中心／ライフスタイル混合 etc.  
- 世界観：清潔感あるシンプル／本音ゴリ押し型／女性的・親近感重視 etc.  
- フォーマット：スライド投稿／文字入り実写／テンプレ活用

→ ✅ **自社に取り入れるべき点：**  
・実績やノウハウを"日常語"で伝えている → 難しさを感じさせない  
・1投稿＝1テーマ完結 → 保存率が高そう

---

#### ② エンゲージメント戦略  
- ストーリーでの巻き込み：アンケート／質問箱／DM誘導の活用  
- コメント欄の使い方：質問投げかけ→会話で関係構築  
- 定期企画：◯◯チャレンジ／限定プレゼント etc.

→ ✅ **自社に取り入れるべき点：**  
・毎週◯曜日のシリーズ企画で"習慣化"している  
・フォロワーとのDMが自然に始まる設計（質問→回答→商品提案）

---

#### ③ プロモーション・CV導線の工夫  
- CTAの使い方：保存促し型／LINE登録誘導／DMで配布  
- プロフィールの構成：1投稿1特典構造／リンクツリー整理  
- 投稿とプロフ、ストーリーの連携性

→ ✅ **自社に取り入れるべき点：**  
・"売る"でなく"選んでもらう"導線設計（例：「これが合う人には神」）  
・プロフィールで「なにが得られるか」を1秒で見せている

---

### 🚀 差別化ポイントの提案（例）

1. **信頼性の裏付け強化**  
　→ 競合は「使ってみた」だけ、自社は「失敗も語る＋比較＋検証」で差別化

2. **データ・検証型コンテンツの導入**  
　→ UGC／レビュー比較／成果報告など、他より"数字と実例"で信頼を高める

3. **ブランドとしての「世界観 × キャラ」戦略**  
　→ キャラ立ちを重視：「副業界の○○」のように人格付け＋共通語を作る

4. **教育ステップ設計型アカウント**  
　→ 単発投稿ではなく「1投稿目→2→3」と成長型の構造にして、継続フォローを促す

---

### 🧠 最終的に目指す状態

| 項目 | 競合 | 自社（提案） |
|------|------|--------------|
| 世界観 | 無機質・ノウハウ系 | ストーリー性・キャラ性重視 |
| CTA | 保存＆LINE登録 | DM・質問からの選別提案型 |
| 投稿構成 | 単発型 | シリーズ化＋教育型導線 |
| 差別化要素 | なし | 失敗体験×選び方ガイド×比較表 |

---

必要であれば、**実際の競合アカウント名**を教えていただければ、リアル分析＋提案を即出力可能です。  
やってみますか？
```

## 使用方法
1. 競合アカウント名または投稿URLを複数収集
2. 自社アカウントのジャンルや方針を明確にする
3. プロンプトを実行して競合分析と差別化戦略を取得

## 活用場面
- 競合他社の調査・分析
- 自社アカウントの差別化戦略立案
- ベンチマーク設定
- マーケティング戦略の見直し
