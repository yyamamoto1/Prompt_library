# Instagram投稿：質問形式＋商品導線

## プロンプト概要
問いかけを軸にした共感型キャプションで、自然に商品やサービスに誘導するプロンプト。

## プロンプト内容

```
あなたはInstagramで、悩みや課題を抱えたターゲット層に向けて、自然に共感と興味を引き出し、商品やサービスに誘導するストーリー性のある投稿を行うプロマーケターです。

以下の情報をもとに、**問いかけを軸にした共感型キャプションと効果的なハッシュタグ**を作成してください。

【入力情報】  
- ターゲット層：[例：副業を始めたい30代サラリーマン／美容に悩む20代女性]  
- 抱える悩み・課題：[例：時間がない／肌荒れが治らない／情報が多すぎて選べない]  
- 解決策となる商品・サービス：[例：AI副業ガイド／敏感肌用スキンケアセット／初心者向け資産運用講座]

【出力形式】
---
📸 **Instagram投稿文（質問形式・共感導入）**

「◯◯で悩んでいませんか？」  
正直、私もずっとそうでした。

☑︎ ◯◯してもうまくいかない  
☑︎ 情報だけ増えて、行動できない  
☑︎ ◯◯するのがしんどい…

そんなとき出会ったのが【◯◯】。  
無理なく、今の自分のままで一歩踏み出せた理由は、  
"◯◯だから"。

もし「私も…」と思ったら、  
この投稿を保存 or プロフィールからチェックしてみてください✨

---

🏷️ **おすすめハッシュタグ（5〜8個）**  
#◯◯で悩んでる人と繋がりたい  
#〇〇初心者  
#〇〇スタート  
#保存したい投稿  
#マイペースに始める  
#自分らしい選択  
#今がチャンス  
#Instagramで学ぶ

---

🧩この投稿スタイルは、「売り込み感なし」で自然に興味とクリックを誘発します。

よければ以下の3点を送っていただければ、実例をすぐ生成します：

- ターゲット層  
- 抱えている悩みや課題  
- 紹介したい商品やサービス  

どうしますか？
```

## 使用方法
1. ターゲット層を具体的に設定
2. 抱える悩み・課題を明確化
3. 解決策となる商品・サービスを記載
4. プロンプトを実行して質問形式の投稿文とハッシュタグを取得

## 活用場面
- 共感を呼ぶ投稿作成
- 売り込み感のない商品紹介
- ターゲット層の絞り込み
- 自然な購買導線の構築
