# Instagramストーリー投稿アイデア＋リンク導線

## プロンプト概要
ストーリー投稿で商品を効果的に紹介し、リンクスタンプ経由での成約を最大化するプロンプト。

## プロンプト内容

```
あなたはInstagramストーリーを活用したセールス設計に精通したプロのマーケターです。  

以下の[紹介する商品やサービス]を、Instagramのストーリー投稿で効果的に紹介し、「リンクスタンプ経由でのクリック・成約」を最大化するための3つのアイデアを提案してください。

【指示内容】
- ストーリーは1投稿で完結せず、**最大3～5枚の構成で提案**
- それぞれ「導入→問題提起→商品紹介→ベネフィット→行動喚起」の流れに沿って構成
- 各アイデアには「リンクスタンプの使い方（貼る位置・誘導文）」もセットで説明

【入力】
- 紹介する商品やサービス：[例：副業スタートガイド／3日間限定セール商品／無料カウンセリング etc.]

【出力形式】
---
📌 **ストーリーアイデア①：〇〇型（例：悩み共感型）**  
- 枚数構成：1枚目（共感）→2枚目（あるある）→3枚目（商品紹介）→4枚目（ベネフィット）→5枚目（リンク誘導）  
- CTA文例：  
　「気になる人はココから✔︎」  
　「3日間だけの特典はここから」  
- リンクスタンプ位置：中央下部にシンプルに／人物と重ならない位置

📌 **ストーリーアイデア②：比較ベネフィット型**  
- Before/After・他商品比較などを挿入  
- リンク誘導は「◯◯と迷ってる人はタップ」型で行動促進

📌 **ストーリーアイデア③：裏技・限定ネタ型**  
- 視聴者が「保存したくなる・知って得する」系のノウハウを先に出し、  
　→ 最後に「そのやり方を手に入れるならコレ」と商品誘導  
- CTAは「◯人が保存中」「この情報、逃すのはもったいない」など

---

🧠このプロンプトは、セールス／アフィリ／無料プレゼント／講座告知など幅広く使えます。

商品名を教えていただければ、すぐに実例を3パターン出しますか？
```

## 使用方法
1. [紹介する商品やサービス]に具体的な商品やサービス内容を記載
2. プロンプトを実行して3つのストーリーアイデアを取得
3. 各アイデアに含まれるリンクスタンプの位置やCTA文を参考にストーリー作成

## 活用場面
- 商品紹介ストーリーの作成
- リンククリック率の向上
- ストーリー投稿の構成設計
- セールス導線の最適化
