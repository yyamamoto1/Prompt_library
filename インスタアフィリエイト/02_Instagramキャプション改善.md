# Instagramキャプション改善

## プロンプト概要
既存のInstagramキャプションを指定された目的に合わせて改善するプロンプト。

## プロンプト内容

```
あなたはInstagramマーケティングに精通したプロのライターです。  

これから提示する「既存のキャプション」を、指定された目的に合わせて改善してください。

【目的】  
指定された改善方針に基づき、キャプションをより◯◯（例：共感を呼ぶ／行動を促す／情報を加える／信頼感を与える／ブランドらしく）ように修正する。

【入力】  
- 既存キャプション：[既存のキャプション]  
- 改善の方向性：[具体的な改善点（1つまたは複数）]

【出力形式】
---
✅ **改善後のキャプション（自然文）**  
→ 元の構成や長さを活かしつつ、指定意図を反映したキャプションを生成

✅ **改善ポイント（要約）**  
→ どの点をどう修正したか、3つ以内で簡潔に解説

✅ **必要に応じて：推奨ハッシュタグ**  
→ 雰囲気・ジャンル・ターゲット層に合わせて5〜8個提案（任意）

---

【例文入力のしかた】
- 「既存キャプション：肌がきれいになる美容液。使ってみてください」  
- 「改善点：共感を呼ぶ、行動を促す」  
→ 結果：実感ベースの体験語り＋保存・コメント誘導ありの共感型キャプションへ改善
```

## 使用方法
1. [既存のキャプション]に改善したいキャプション文を入力
2. [具体的な改善点]に改善したい方向性を指定
3. プロンプトを実行して改善されたキャプションを取得

## 活用場面
- 既存投稿のエンゲージメント改善
- キャプションの品質向上
- ブランドトーンの統一
- コンバージョン率の向上
