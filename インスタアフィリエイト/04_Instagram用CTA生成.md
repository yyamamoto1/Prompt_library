# Instagram用 行動喚起文 CTA生成

## プロンプト概要
プロフィールリンクへのクリックを促す効果的な行動喚起文（CTA）を生成するプロンプト。

## プロンプト内容

```
あなたはInstagramマーケティングに長けたプロのコピーライターです。  

以下の内容に基づき、[紹介する商品やサービス]に興味を引きつけ、プロフィールリンクへのクリックを促す「行動喚起文（CTA）」を3パターン作成してください。

【目的】  
- 興味喚起からクリックまでの導線をつなげる  
- 「緊急性」や「限定感」を含め、今すぐ行動したくなる心理を刺激する  
- ストーリーズ・フィード投稿・リールでも使えるよう、短く強い文にする

【入力】
- 紹介する商品・サービス：[例：AIブログ講座、初回無料カウンセリング、3日間限定のLINE登録特典付きキャンペーン など]

【出力形式】
---
📝 **CTAパターン1（王道）**  
→  
📣 **CTAパターン2（感情訴求）**  
→  
⚡ **CTAパターン3（緊急＆限定強調）**  
→  
```

## 使用方法
1. [紹介する商品・サービス]に具体的な商品やサービス内容を記載
2. プロンプトを実行して3つのCTAパターンを取得
3. 投稿の目的やターゲットに応じて最適なCTAを選択

## 活用場面
- プロフィールリンクへの誘導
- 成約率の向上
- 緊急性のあるキャンペーン告知
- ストーリーズでのリンクスタンプ活用
