# Instagram投稿頻度＆タイミング最適化

## プロンプト概要
Instagramアフィリエイトの成果を最大化する投稿頻度と時間帯を提案するプロンプト。

## プロンプト内容

```
あなたはInstagramアフィリエイトに特化したSNS戦略コンサルタントです。  

以下の情報をもとに、[アカウントのテーマ]と[ターゲット層]に最適な「投稿頻度」と「時間帯」を提案してください。

【目的】  
Instagramでのアフィリエイト成果を最大化するために、  
- 投稿の露出（リーチ）を増やす  
- 保存・シェアされる投稿を適切なタイミングで届ける  
- ターゲット層の"アクティブ時間"を活かす

【入力】  
- アカウントのテーマ：[例：副業ノウハウ／美容アイテム紹介／時短ガジェット／投資・資産形成]  
- ターゲット層：[例：20代女性／会社員男性／育児中のママ／30〜40代の副業層]

【出力形式】
---
📆 **理想的な投稿頻度**
- フィード投稿：◯回／週（理由つき）
- ストーリーズ：◯回／日（＋活用法）
- リール投稿（短尺動画）：◯回／週（優先すべき内容）

🕐 **最適な投稿時間帯（曜日別 or 一般傾向）**
- 平日：◯時〜◯時（例：通勤・昼休憩・就寝前）  
- 土日：◯時〜◯時（例：家事後・ゆったり時間）  
※「ストーリー・リール」は朝活／夜活など視聴習慣に応じて調整

🎯 **補足アドバイス**
- 保存されやすい投稿の曜日とタイミング  
- 商品紹介系は◯曜日／悩み共感系は◯曜日が効果的  
- ストーリー×リンクスタンプの使いどき提案  
- フィード→ストーリー→リールの連動導線の作り方（必要に応じて）

---

このプロンプトなら、**ターゲットの"生活時間に合わせた自然な接触"を作れます**。

よければ、以下の2点を送ってください：
- アカウントのテーマ  
- ターゲット層（属性・生活パターン）

実例を即、最適化してご提案します。どうしますか？
```

## 使用方法
1. アカウントのテーマを明確にする
2. ターゲット層の属性と生活パターンを分析
3. プロンプトを実行して投稿頻度と最適タイミングを取得
4. 提案内容に基づいて投稿スケジュールを設計

## 活用場面
- 投稿スケジュールの最適化
- エンゲージメント率向上
- ターゲット層へのリーチ拡大
- アフィリエイト成果の最大化
