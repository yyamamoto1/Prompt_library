# Instagram投稿：体験談ベースの紹介文

## プロンプト概要
体験談を軸にした共感性の高いInstagram投稿文を作成するプロンプト。

## プロンプト内容

```
あなたはInstagramで影響力のあるリアルな体験発信者です。  

以下の[自身が体験した商品やサービス]について、**具体的なエピソードを交えながら、共感・信頼・興味を生む紹介文**を作成してください。

【目的】  
- 読者が「私も同じだった」「試してみたい」と感じるような体験ベースの紹介にする  
- 押し売り感を出さず、自然にメリットが伝わる構成にする  
- 見た人が「保存」「シェア」したくなるような温度感と共感性を重視する

【入力】  
- 商品／サービス名：[例：AIライティング講座／オーガニックナイトクリーム]  
- 自身の体験談（どんな悩み or 状況 → どう使った → どんな変化があったか）

【構成ルール】  
1. **導入：過去の悩み・状況（共感される一言）**  
2. **体験：商品との出会い／使ってみた実感／印象的な変化**  
3. **まとめ：これからの展望 or 同じ悩みを持つ人へのひとこと**  
4. **行動喚起：保存・チェック・DMなどの軽いCTA**

【出力形式】
---
📸 **Instagram紹介文（体験ベース）**

「正直、〇〇にはずっと悩んでました。」  
そんなときに出会ったのが【◯◯】。

最初は半信半疑だったけど、△△を使い始めたら  
☑︎ ◯◯がラクにできた  
☑︎ △△の不安が消えた  
☑︎ 朝起きたときの◯◯がまるで別人レベル…！

今では「これ、もっと早く知りたかった」って心底思ってます。  
同じように悩んでたら、ぜひ一度チェックしてみて😊  

気になる人は、ストーリーのリンク or プロフィールから✔︎

---

このプロンプトを使えば、「売らずに売れる体験型セールス投稿」が量産できます。

✅ 商品名  
✅ あなたのリアルな体験  
を送っていただければ、すぐ1本出力可能です。準備OKですか？
```

## 使用方法
1. [商品／サービス名]に紹介したい商品を記載
2. 自身の体験談（悩み→使用→変化）を具体的に記載
3. プロンプトを実行して体験談ベースの紹介文を取得

## 活用場面
- 信頼性の高い商品紹介
- 共感を呼ぶ投稿作成
- 売り込み感のない自然な訴求
- ファンづくりとエンゲージメント向上
