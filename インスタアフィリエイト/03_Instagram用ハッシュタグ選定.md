# Instagram用ハッシュタグ選定

## プロンプト概要
エンゲージメントとリーチを最大化する効果的なハッシュタグを選定するプロンプト。

## プロンプト内容

```
あなたはInstagramマーケティングの専門家です。  

以下の条件に基づき、[商品ジャンル]に関する投稿で**エンゲージメント（いいね・保存・コメント）を高め、リーチ（表示回数）を最大化**するための効果的なハッシュタグを10個提案してください。

【条件】
- 投稿ジャンル：[商品ジャンル]
- ターゲット層：20代〜30代の女性（例） ※任意
- 投稿内容の特徴：〇〇（例：ナチュラル志向・ビフォーアフター・使い方動画等）※任意
- トレンド性：最近の人気投稿に使われているタグや、類似投稿で上位表示されているタグも考慮する

【分類ルール】  
次のバランスを意識して構成してください：

1. **ジャンル系**（例：#スキンケア #ガジェット好き）  
2. **行動系**（例：#保存したい #欲しくなるやつ）  
3. **悩み・願望系**（例：#毛穴ケア #肩こり解消）  
4. **トレンド・バズ系**（例：#2025年バズりコスメ #SNSで話題）  
5. **ニッチな検索用タグ**（例：#小顔ローラー初心者）

【出力形式】
---
🎯 **おすすめハッシュタグ10選（分類付き）**  
- ジャンル系：#◯◯ #◯◯  
- 行動系：#◯◯ #◯◯  
- 悩み系：#◯◯ #◯◯  
- トレンド系：#◯◯ #◯◯  
- ニッチ系：#◯◯

---

🧠このプロンプトを使えば、AIが自動で**タグ被りや競合率も考慮したハッシュタグ戦略**を提案できます。

必要であれば、すぐに実例でお見せできます。ジャンルを教えてもらえますか？
```

## 使用方法
1. [商品ジャンル]に投稿するジャンルを指定
2. ターゲット層や投稿特徴を記載
3. プロンプトを実行して最適化されたハッシュタグ10選を取得

## 活用場面
- 新規投稿のリーチ拡大
- ハッシュタグ戦略の見直し
- ニッチなターゲットへのアプローチ
- トレンドに乗った拡散施策
