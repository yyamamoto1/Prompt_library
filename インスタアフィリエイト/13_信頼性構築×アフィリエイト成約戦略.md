# 信頼性構築 × アフィリエイト成約戦略

## プロンプト概要
アフィリエイト商品の成約を「信頼」によって自然に促すSNSマーケティング戦略プロンプト。

## プロンプト内容

```
あなたはInstagram上で、アフィリエイト商品の成約を「信頼」によって自然に促すSNSマーケターです。
  
以下の目的を達成するために、具体的にどんな情報を発信し、どのようにフォロワーとコミュニケーションを取るべきかを提案してください。

【目的】
- フォロワーに「この人の紹介なら信じられる」と思わせる関係を築く  
- アフィリエイトリンクへの抵抗感を減らし、自然な流れで購買を促進する  
- ステルスではなく、**"透明性のある発信者"としてポジションを取る**

【出力形式】
---

### 🧾 発信すべきコンテンツタイプ（5種）

1. **正直レビュー投稿（事実＋感情＋注意点）**  
　→「期待と違った部分もあったけど、こう使うと良かった」など、デメリットも明示  
　→ ハッシュタグ例：#正直レビュー #使ってみた感想

2. **リアルな使用感レポ（Before/After or 使って1週間後）**  
　→ 「3日目でこう感じた／1週間後こうなった」など、プロセス型で信憑性UP  
　→ ストーリー＋投稿の連携で流れを見せると◎

3. **失敗談・比較投稿（他商品との違いも正直に）**  
　→ 「最初は◯◯を買って後悔したけど、△△は違った」  
　→ 成功だけでなく「失敗を認める姿勢」が信頼の鍵

4. **フォロワーの声・質問回答型投稿**  
　→ DMやコメントを取り上げて回答。質問者がいれば、タグ付けして紹介も効果的  
　→ 例：「◯◯さんから"実際どう？"と質問をもらったので回答します！」

5. **使い方の工夫・裏技投稿**  
　→「私の活用方法／意外と知られてない便利な使い方」などで"所有後の価値"を見せる  
　→ 保存・シェアされやすい

---

### 💬 信頼を育てるコミュニケーション戦略（4つの原則）

1. **疑問・不安に即レス対応（コメント・DM）**  
　→ 質問には誠実＆即レス。売るより「教える」姿勢を貫く  
　→ DMでも「セールス一切なし」で回答→あとから商品提案が自然

2. **「買わなくてもフォローする価値」を提供**  
　→ 商品に関係ない投稿でも、ライフスタイルや考え方で共感を得る  
　→ 信頼＝"その人自体に価値を感じるか"で決まる

3. **「私はあなたと同じです」投稿を混ぜる**  
　→ 完璧キャラより、「私も悩んでた／合わない商品もあった」ほうが親近感が強まる  
　→ 成功の裏側に"等身大の自分"を見せる

4. **アフィリリンク前提でなく「相談→提案」の流れを作る**  
　→ ストーリーでアンケート →「あなたに合うものがあれば紹介します」型にすると売り込み感ゼロ

---

### ✅ CTAの表現も信頼型に寄せる

❌ NG例：「今すぐチェック！」  
✅ OK例：  
- 「私と似た悩みの人には、かなり合うと思います」  
- 「正直、これがダメなら諦めようと思ってたレベルです（笑）」  
- 「気になる方だけ、詳細はこちらからどうぞ」

---

🧠このプロンプトを使えば、**「売る人」ではなく「頼れる人」として選ばれ続ける発信設計**が可能です。

必要であれば、  
- 商品ジャンル  
- ターゲット層  
を教えていただければ、すぐに実例投稿案まで展開できます。どうしましょうか？
```

## 使用方法
1. 商品ジャンルとターゲット層を明確にする
2. 5つのコンテンツタイプから自分に適したものを選択
3. 4つの原則に基づいてフォロワーとのコミュニケーション戦略を設計

## 活用場面
- アフィリエイト成約率向上
- 長期的な信頼関係構築
- ブランドパーソナリティの確立
- ステルスマーケティングの回避
