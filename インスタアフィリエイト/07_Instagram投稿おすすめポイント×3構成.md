# Instagram投稿：おすすめポイント×3構成

## プロンプト概要
商品の「おすすめポイントを3つ」で構成した魅力的なInstagram投稿文を作成するプロンプト。

## プロンプト内容

```
あなたはInstagramマーケティングのプロです。  

以下の情報をもとに、[具体的な商品名]の「おすすめポイントを3つ」紹介する投稿文を作成してください。

【目的】  
- 指定のターゲット層に刺さる言葉で構成する  
- 各おすすめポイントに魅力的な言葉を添え、読者の共感・行動を促す  
- 読後に「欲しい」「使ってみたい」と思わせる仕上がりにする

【入力】  
- 商品名：[具体的な商品名]  
- ターゲット層：[例：30代女性／美容好き／時短グッズを探している人 など]

【構成ルール】  
1. **導入：悩み・共感・驚きの一言で惹きつける**  
2. **本文：おすすめポイントを1つずつ展開（合計3つ）**  
3. **締め：行動促進の一言（保存・シェア・リンク誘導など）**

【出力形式】
---
📸 **Instagram投稿文（本文）**

「◯◯で悩んでませんか？」  
そんなあなたにぴったりなのが【〇〇】。  

🔸おすすめポイント①：〇〇（→ 説明・ベネフィット）  
🔸おすすめポイント②：〇〇（→ 説明・ベネフィット）  
🔸おすすめポイント③：〇〇（→ 説明・ベネフィット）

「これ知れて良かった！」と思ったら保存してあとで見返してね✍️  
気になる人はプロフィールのリンクからチェック✅

---

💡この構成なら、**セールス色が強すぎず、自然に「いいね・保存・クリック」を引き出せます。**

実例を出せますので、  
- 商品名  
- ターゲット層  
を送ってください。すぐ作成可能です！
```

## 使用方法
1. [具体的な商品名]に紹介したい商品名を記載
2. [ターゲット層]に想定するターゲット層を記載
3. プロンプトを実行して3つのおすすめポイントで構成された投稿文を取得

## 活用場面
- 商品の魅力を体系的に紹介
- 保存されやすい投稿の作成
- セールス色を抑えた自然な商品紹介
- エンゲージメント向上
