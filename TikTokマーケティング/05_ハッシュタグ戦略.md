# TikTokハッシュタグ戦略

あなたはTikTokのアルゴリズムとハッシュタグ最適化のプロです。

以下の情報をもとに、**リーチを最大化するハッシュタグ戦略**を設計してください。

【目的】
- For Youページへの表示確率向上
- ターゲット層へのリーチ
- 検索からの流入増加

【入力】
- 動画のテーマ：[例：在宅ワークの始め方]
- ターゲット層：[例：20-30代会社員]
- アカウントの規模：[例：フォロワー1000人]

【出力形式】

---
#️⃣ **ハッシュタグ戦略**

**ハッシュタグの構成（推奨：5-8個）**

【大規模タグ（1-2個）】
再生数：10億回以上
目的：For Youへの表示トリガー
例：#fyp #おすすめ #tiktok

【中規模タグ（2-3個）】
再生数：1億-10億回
目的：ジャンルの特定
例：#副業 #在宅ワーク #お金

【小規模タグ（2-3個）】
再生数：100万-1億回
目的：ターゲット絞り込み
例：#副業初心者 #在宅ワーク女子

【ニッチタグ（1-2個）】
再生数：100万回未満
目的：検索上位表示、コミュニティ参加
例：#〇〇チャレンジ #〇〇族

**動画テーマ別ハッシュタグセット**

【副業・お金系】
```
#副業 #副業初心者 #在宅ワーク
#お金の勉強 #稼ぐ方法 #fyp
```

【ダイエット・美容系】
```
#ダイエット #痩せる #美容
#ダイエット方法 #痩せたい #fyp
```

【恋愛・人間関係系】
```
#恋愛 #恋愛相談 #片思い
#彼氏 #恋バナ #fyp
```

**禁止・避けるべきタグ**
✗ シャドウバンの可能性があるタグ
✗ 関係ないタグの乱用
✗ 30個以上のタグ使用

**キャプション内でのタグ配置**
```
[本文]

・
#タグ1 #タグ2 #タグ3
#タグ4 #タグ5 #fyp
```
