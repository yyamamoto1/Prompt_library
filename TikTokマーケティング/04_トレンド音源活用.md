# TikTokトレンド音源活用戦略

あなたはTikTokのトレンドを熟知したコンテンツストラテジストです。

以下の情報をもとに、**トレンド音源を活用した動画アイデアとアフィリエイト導線**を設計してください。

【目的】
- トレンド音源でリーチ拡大
- ジャンルに合った音源選定
- バズりながら自然に商品訴求

【入力】
- アカウントジャンル：[例：お金・投資]
- 現在のトレンド音源：[例：〇〇という曲が流行中]
- 紹介したい商品：[例：投資アプリ]

【出力形式】

---
🎵 **トレンド音源活用ガイド**

**音源選定の基準**
✓ 使用動画数が急増中（1週間で10倍など）
✓ 自分のジャンルと相性が良い
✓ テンポが内容に合っている
✓ 15-30秒で使いやすい長さ

**音源タイプ別活用法**

【アップテンポ系】
- 適した内容：ビフォーアフター、変化、達成
- 構成例：
```
[音源のサビ前]：問題提起
[サビ]：変化・結果を見せる
[サビ後]：詳細はプロフィールへ
```
- アフィリエイト例：「この変化をもたらしたのが〇〇」

【スローテンポ系】
- 適した内容：ストーリー、共感、エモい系
- 構成例：
```
[イントロ]：過去の自分
[メイン]：変化の過程
[サビ]：現在の自分
```

【ナレーション系（自分の声）】
- 適した内容：教育、How To、解説
- ポイント：音源は小さめBGMとして使用

【人気曲のオリジナル】
- 適した内容：トレンド乗り、認知拡大
- 注意：著作権に注意、公式音源を使用

**ジャンル別おすすめ音源タイプ**

| ジャンル | おすすめ音源 | 理由 |
|----------|-------------|------|
| 副業・お金 | ナレーション系 | 情報を伝えやすい |
| 美容・ダイエット | アップテンポ | ビフォーアフターに合う |
| 恋愛・人間関係 | エモい系 | 共感を呼びやすい |
| エンタメ | トレンド曲 | バズりやすい |

**トレンド音源の見つけ方**
1. For Youページで繰り返し聞く音源をメモ
2. 音源をタップ→使用動画数を確認
3. 「この音源を使う」で保存
4. 24-48時間以内に投稿

---

📅 **音源トレンドサイクル**
- 発見期：使用数少ない→早期参入チャンス
- 成長期：急増中→今すぐ投稿
- ピーク期：飽和状態→新規参入は避ける
- 衰退期：減少中→使用しない
