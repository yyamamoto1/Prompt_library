# TikTok法令遵守・コミュニティガイドライン

あなたはTikTokの規約と関連法規に詳しい専門家です。

以下の情報をもとに、**アカウントを守りながら収益化するためのガイドライン**を作成してください。

【目的】
- アカウントBAN・シャドウバンの防止
- 法令違反の回避
- 長期的に安全な運営

【入力】
- アカウントジャンル：[例：美容レビュー]
- 紹介する商品カテゴリ：[例：化粧品、サプリ]
- 収益化方法：[例：アフィリエイト、企業案件]

【出力形式】

---
⚖️ **TikTok運営ガイドライン**

**1. ステルスマーケティング規制（景品表示法）**

【2023年10月〜 義務化】
- 広告・PRであることの明示が必須
- 違反すると法的リスクあり

【表記方法】
① キャプション
```
【PR】[企業名]
または
#PR #広告 #タイアップ
```

② 動画内テロップ
```
冒頭に「PR」「広告」を表示
```

③ 音声
```
「この動画は〇〇とのタイアップです」
```

**表記が必要なケース**
✓ 企業から報酬をもらった投稿
✓ 商品を無料でもらったレビュー
✓ アフィリエイトリンクを含む投稿（推奨）

**2. 薬機法対応（健康・美容系）**

【禁止表現】
✗ 「これで病気が治る」
✗ 「確実に痩せる」「絶対効く」
✗ 医薬品的な効能効果の表現
✗ ビフォーアフターの過度な誇張

【OK表現】
✓ 「私は〇〇を感じました（個人の感想）」
✓ 「〇〇をサポート」
✓ 「※効果には個人差があります」

**3. TikTokコミュニティガイドライン**

【禁止コンテンツ】
✗ 危険行為の助長
✗ ヘイトスピーチ
✗ 誤情報・デマ
✗ スパム行為
✗ 著作権侵害

【シャドウバンのリスク要因】
⚠️ 同じ動画の繰り返し投稿
⚠️ 過度なハッシュタグ使用
⚠️ 他SNSのウォーターマーク
⚠️ 規約違反コンテンツ
⚠️ 短時間での大量投稿

**4. 著作権対応**

【音源】
✓ TikTok内の音源ライブラリ
✓ 著作権フリー音源
✗ 無断使用の市販楽曲

【画像・映像】
✓ 自分で撮影した素材
✓ 商用利用可能な素材
✗ 他者のコンテンツの無断転載

**5. 年齢制限コンテンツ**

【注意が必要なカテゴリ】
- アルコール関連
- ギャンブル関連
- 性的な示唆を含む内容
