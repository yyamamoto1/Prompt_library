# TikTok動画台本テンプレート

あなたはTikTokで高いエンゲージメントを獲得する台本作成のプロです。

以下の情報をもとに、**最後まで見られる動画台本**を作成してください。

【目的】
- 視聴完了率の最大化
- コメント・シェアの促進
- プロフィールへの誘導

【入力】
- 動画の目的：[例：商品紹介、教育、エンタメ]
- 動画の長さ：[例：30秒/60秒]
- 伝えたいメッセージ：[例：副業で月5万円稼ぐ方法]
- 紹介する商品：[例：副業スクール]

【出力形式】

---
📝 **動画台本（30秒版）**

```
【0-3秒：フック】
「まだ知らないの？」
「これ見て人生変わった」
「99%の人が間違えてる」

【3-8秒：問題提起/共感】
「副業始めたいけど何からやればいいかわからない、
そんな人多いと思う」

【8-23秒：メインコンテンツ】
「結論から言うと〇〇」
「ポイントは3つ」
「①〇〇」
「②〇〇」
「③〇〇」

【23-27秒：ベネフィット】
「これやるだけで〇〇」

【27-30秒：CTA】
「詳しくはプロフィールから」
「保存して後で見返して」
「続きはパート2で」
```

---

📝 **動画台本（60秒版）**

```
【0-3秒：フック】
[衝撃的な一言/質問]

【3-10秒：自己紹介＋権威づけ】
「〇〇を達成した私が教える」

【10-15秒：問題提起】
「こんな悩みありませんか？」

【15-45秒：解決策/価値提供】
[ステップバイステップで解説]

【45-55秒：商品紹介】
「私が使ったのはこれ」
「詳しくはプロフィールのリンクから」

【55-60秒：CTA】
「フォローして他の投稿もチェック」
「質問はコメントで」
```

---

🎤 **話し方のポイント**
- テンポよく、間を空けすぎない
- 感情を込める（驚き、共感、興奮）
- 視聴者に語りかける口調
- 「あなた」「みんな」を使う
