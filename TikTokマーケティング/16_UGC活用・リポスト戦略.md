# TikTok UGC活用・リポスト戦略

あなたはUGC（ユーザー生成コンテンツ）マーケティングのプロです。

以下の情報をもとに、**UGCを活用してアカウント成長と収益化を加速する戦略**を設計してください。

【目的】
- 視聴者参加型コンテンツの創出
- 信頼性・社会的証明の強化
- コンテンツ制作コストの削減

【入力】
- アカウントジャンル：[例：フィットネス]
- フォロワー数：[例：1万人]
- 紹介している商品：[例：プロテイン、トレーニングウェア]

【出力形式】

---
📱 **UGC活用戦略**

**UGCの種類と活用法**

① レビュー・体験談系
- 内容：商品を使った視聴者の声
- 活用：デュエットでコメント付けてリポスト
- 効果：社会的証明、信頼性UP

② チャレンジ参加系
- 内容：指定のハッシュタグでチャレンジ投稿
- 活用：ベストを選んでフィーチャー
- 効果：認知拡大、エンゲージメント向上

③ 質問・相談系
- 内容：フォロワーからの質問
- 活用：動画で回答
- 効果：双方向コミュニケーション強化

**UGCを集める仕組み**

【ハッシュタグチャレンジ】
```
「#〇〇チャレンジ で投稿してね！
ベストな投稿はこのアカウントで紹介します✨
さらに抽選で[特典]をプレゼント🎁」
```

【商品レビュー募集】
```
「[商品名]使ってみた人！
感想を動画で投稿してくれたら
このアカウントで紹介するよ📱
#〇〇レビュー を付けてね」
```

【質問募集】
```
「[テーマ]について質問ある人！
コメントかDMで教えて📩
動画で回答します」
```

**UGCリポストの方法**

① デュエットでリポスト
- 元動画 + 自分のリアクション
- コメント・解説を追加
- 元投稿者をタグ付け

② Stitchでリポスト
- 元動画の一部を引用
- 自分の補足を追加
- より詳しく解説

③ まとめ動画
- 複数のUGCをまとめて紹介
- 「みんなの〇〇」形式
- ベストセレクション

**UGC活用時の注意点**
✓ 必ず許可を得る（DMで確認）
✓ 元投稿者をクレジット
✓ ポジティブなコンテンツを選ぶ
✗ 無断転載は絶対NG
✗ ネガティブな内容は避ける
