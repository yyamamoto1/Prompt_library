**コメント対応の基本方針**

| コメント種類 | 対応方針 | 返信例 |
|-------------|----------|--------|
| 質問 | 丁寧に回答 | 「ご質問ありがとうございます！[回答]」 |
| 称賛 | 感謝を伝える | 「嬉しいです！ありがとうございます😊」 |
| 建設的批判 | 真摯に受け止め | 「貴重なご意見ありがとうございます」 |
| 誹謗中傷 | 無視or削除 | 返信しない |
| 荒らし | ブロック | 対応しない |

**よくあるネガティブコメントへの返信テンプレート**

① 「ステマでしょ？」「PR案件？」
```
ご指摘ありがとうございます。
PRの場合は必ず明記しております。
今回は自分で購入して本当に良かったので
紹介させていただきました😊
```

② 「効果なかった」「買って損した」
```
お気持ちお察しします。
効果には個人差がありますよね。
どのような点が期待と違いましたか？
参考にさせていただきます🙏
```

③ 「嘘つき」「信用できない」
```
ご不快な思いをさせてしまい申し訳ありません。
私の個人的な感想としてお伝えしており、
効果を保証するものではございません。
今後はより丁寧にお伝えできるよう心がけます。
```

④ 「金儲け目的」「アフィカス」
```
ご意見ありがとうございます。
本当に良いと思ったものだけを
紹介するようにしています。
不快に感じられたら申し訳ありません。
```

【投稿前チェック】
☐ 誇大表現はないか

☐ PR表記は適切か

【炎上しやすいトピック（注意）】
⚠️ 政治・宗教

**炎上時の対応フロー**

```
炎上発生
  ↓
①状況把握（何が問題か）
  ↓
②初動対応（24時間以内）
  - 謝罪が必要：誠実に謝罪
  - 誤解：事実を説明
  ↓
③沈静化を待つ
  - 追加投稿は控える
  - 煽りに反応しない
  ↓
④再発防止策の検討
```
