# TikTok商品紹介動画テンプレート

あなたはTikTokでの商品紹介で高いコンバージョンを出すプロです。

以下の情報をもとに、**押し売り感なく商品の魅力を伝える動画台本**を作成してください。

【目的】
- 商品への興味喚起
- プロフィールリンクへの誘導
- 購入意欲の向上

【入力】
- 商品名：[例：〇〇美容液]
- 商品の特徴：[例：保湿力が高い、敏感肌でも使える]
- ターゲット：[例：肌荒れに悩む20代女性]
- 動画の長さ：[例：30秒]

【出力形式】

---
📦 **商品紹介台本パターン**

**パターン①：Before/After型（30秒）**
```
[0-3秒] フック
「この商品に出会って人生変わった」

[3-10秒] Before（問題提起）
「ずっと〇〇で悩んでた」
[悩んでいた時の映像/写真]

[10-20秒] 商品紹介
「で、この[商品名]使い始めて」
[商品を見せる、使い方を見せる]

[20-27秒] After（結果）
「今はこう」
[変化した状態を見せる]

[27-30秒] CTA
「気になる人はプロフィールから」
```

**パターン②：開封レビュー型（30秒）**
```
[0-3秒] フック
「ずっと気になってたやつ買ってみた」

[3-15秒] 開封
[実際に開封する様子]
「パッケージ可愛い」「香りは〇〇」

[15-25秒] 使用感
「使ってみた感想は...」
[実際に使用]

[25-30秒] まとめ＋CTA
「正直[評価]！リンクはプロフに」
```

**パターン③：比較型（30秒）**
```
[0-3秒] フック
「[商品A]と[商品B]どっちがいい？」

[3-15秒] 商品A紹介
[特徴、メリデメ]

[15-25秒] 商品B紹介
[特徴、メリデメ]

[25-30秒] 結論
「私的には[商品]！詳しくはプロフ」
```

**パターン④：使い方解説型（60秒）**
```
[0-3秒] フック
「[商品]の正しい使い方知ってる？」

[3-50秒] ステップ解説
STEP1：[手順]
STEP2：[手順]
STEP3：[手順]

[50-60秒] CTA
「詳しい情報はプロフィールから」
```

---

💡 **商品紹介のコツ**
- 「PR」と明記（必須）
- 自分の言葉で正直に
- デメリットも少し触れる（信頼性UP）
- 実際に使っている様子を見せる
