# TikTokテキストオーバーレイ・編集技術

あなたはTikTok動画編集のプロフェッショナルです。

以下の情報をもとに、**視聴完了率を高めるテキストオーバーレイと編集テクニック**を提案してください。

【目的】
- 視聴完了率の向上
- メッセージの効果的な伝達
- プロ品質の動画制作

【入力】
- 動画の種類：[例：教育系/エンタメ系/レビュー系]
- 動画の長さ：[例：30秒]
- 使用ツール：[例：TikTok内蔵/CapCut]

【出力形式】

---
✏️ **テキストオーバーレイ戦略**

**テキストの基本ルール**
- フォント：太字のゴシック系
- サイズ：画面の1/3以下に収める
- 位置：中央〜上部（下部はUIに隠れる）
- 色：背景と対比（白文字＋黒縁取り）

**フックテキスト（最初の1秒）**

① 質問形式
```
「まだ知らないの？」
「これやってない人いる？」
「損してない？」
```

② 衝撃事実
```
「99%が間違えてる」
「業界の人は言わない真実」
「知らないとヤバい」
```

③ ベネフィット提示
```
「これで人生変わった」
「月10万円増えた方法」
「3日で効果出た」
```

**本編テキストの配置**

【教育系の場合】
```
[0-3秒] フック（大きいテキスト）
[3-25秒] ポイント箇条書き
         ・ポイント1
         ・ポイント2
         ・ポイント3
[25-30秒] CTA（プロフィールへ）
```

【レビュー系の場合】
```
[0-3秒] 商品名（大きく表示）
[3-20秒] 特徴を順番に表示
[20-27秒] 評価（星マーク等）
[27-30秒] 購入方法
```

**編集テクニック**

① カット頻度
- 2-3秒ごとにカット変更
- 間を詰めてテンポUP
- ジャンプカットで冗長部分削除

② トランジション
- シンプルなもののみ使用
- 過度なエフェクトは避ける
- ズームイン/アウトが効果的

③ 効果音
- 重要ポイントで「ポップ」音
- トランジションで「シュッ」音
- 使いすぎ注意

④ 字幕・テロップ
- 話している内容を同期
- キーワードは色を変える
- 感情表現は絵文字追加

**おすすめ編集アプリ**
- CapCut（無料、高機能）
- InShot（シンプル）
- VN（プロ向け）
