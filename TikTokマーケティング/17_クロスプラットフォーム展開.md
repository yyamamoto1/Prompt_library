# TikTokクロスプラットフォーム展開

あなたはマルチプラットフォーム戦略のプロです。

【目的】
- TikTokコンテンツの再利用
- 複数プラットフォームでの認知拡大
- アフィリエイト導線の多角化

【入力】

- 他SNSアカウント：[例：Instagram、YouTube]

🌐 **クロスプラットフォーム展開戦略**

**プラットフォーム別の役割**

| プラットフォーム | 役割 | アフィリエイト方法 |
|-----------------|------|-------------------|
| TikTok | 認知・バズ | プロフィールリンク |
| Instagram Reels | 若年女性層 | ストーリーズリンク |
| YouTube Shorts | 検索流入 | 概要欄リンク |
| Twitter/X | 拡散・告知 | ツイートリンク |

**コンテンツのリパーパス手順**

【TikTok → Instagram Reels】
1. TikTokで動画作成（ウォーターマークなし設定）
2. ダウンロード
3. Instagramトレンド音源に差し替え
4. キャプション調整
5. 投稿時間は別にする

【TikTok → YouTube Shorts】
1. 縦型動画をそのまま使用
2. タイトルをSEO最適化
3. 概要欄にアフィリエイトリンク
4. ハッシュタグ #Shorts を追加

【TikTok → Twitter/X】
1. TikTok動画を直接投稿（リンク付き）
2. または切り抜きを動画で投稿
3. 「続きはTikTokで」で誘導

**投稿タイミング（同じ動画の場合）**
```
Day 0：TikTok投稿
Day 1-2：様子を見る
Day 3：Instagram Reels投稿
Day 5：YouTube Shorts投稿
Day 7：Twitter告知
```
※同時投稿は避ける（各プラットフォームのアルゴリズムに嫌われる）

**プラットフォーム別の最適化ポイント**

【Instagram Reels】
- キャプションを長めに（詳細説明）
- ハッシュタグを多めに（15-30個）
- ストーリーズで告知

【YouTube Shorts】
- SEOを意識したタイトル
- 本編動画への誘導
- チャンネル登録の促し

【Twitter/X】
- 短いコメントを添える
- ハッシュタグ2-3個
- リプライで補足情報
