# TikTokシリーズ・連載企画設計

あなたはTikTokでシリーズコンテンツを成功させるプロです。

以下の情報をもとに、**フォロワーを増やし継続視聴を促すシリーズ企画**を設計してください。

【目的】
- フォロー率の向上（続きを見たい）
- 視聴習慣の形成
- アフィリエイト商品への段階的誘導

【入力】
- ジャンル：[例：投資初心者向け]
- ターゲット：[例：20代会社員]
- 紹介したい商品：[例：証券口座]

【出力形式】

---
📺 **シリーズ企画設計**

**シリーズ①：ストーリー型（5-10本）**
「私が〇〇を達成するまで」

構成：
```
Part1：「借金100万だった私が」(過去)
Part2：「出会ったきっかけ」
Part3：「最初にやったこと」
Part4：「壁にぶつかった話」
Part5：「転機が訪れた」
Part6：「結果こうなった」(現在)
Part7：「使ったツール紹介」→アフィリエイト
```

効果：感情移入→信頼構築→商品購入

**シリーズ②：教育型（7-14本）**
「初心者向け〇〇完全攻略」

構成：
```
Day1：「〇〇とは？基礎の基礎」
Day2：「まず準備するもの」→アフィリエイト
Day3：「最初の一歩」
Day4：「よくある間違い」
Day5：「中級テクニック」
Day6：「上級者への道」
Day7：「まとめ＆おすすめツール」→アフィリエイト
```

効果：価値提供→権威性確立→商品購入

**シリーズ③：ランキング型（継続）**
「〇〇ランキング毎週更新」

構成：
```
Week1：「おすすめ〇〇TOP5」
Week2：「コスパ最強〇〇TOP5」
Week3：「初心者向け〇〇TOP5」
...
```

効果：定期視聴の習慣化→フォロー→商品購入

**シリーズ④：Q&A型（継続）**
「フォロワーの質問に答えるシリーズ」

構成：
```
Q&A#1：「〇〇について」
Q&A#2：「△△について」
...
```

効果：双方向コミュニケーション→信頼構築

**シリーズ成功のポイント**

① タイトルの統一感
- 「【Part○】〇〇シリーズ」
- 「Day○：〇〇」
- 「Vol.○」

② サムネイル（表紙）の統一感
- 同じフォーマット
- 番号を大きく表示
- シリーズカラーを決める

③ 適切な長さ
- 1本あたり15-60秒
- 長すぎると離脱

④ 次回予告
- 「続きはPart2で」
- 「フォローして見逃さないで」
