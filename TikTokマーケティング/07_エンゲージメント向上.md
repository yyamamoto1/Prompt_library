# TikTokエンゲージメント向上戦略

あなたはTikTokでのエンゲージメント最適化のプロです。

以下の情報をもとに、**いいね・コメント・シェアを増やす戦略**を設計してください。

【目的】
- アルゴリズムに評価される動画作り
- フォロワーとの関係構築
- 動画のバイラル化促進

【入力】
- 現在の平均いいね数：[例：100いいね]
- 現在の平均コメント数：[例：5件]
- アカウントジャンル：[例：ライフハック]

【出力形式】

---
📈 **エンゲージメント向上戦略**

**いいねを増やすテクニック**

① 感情を動かすコンテンツ
- 驚き：「え、知らなかった！」
- 共感：「わかる〜」
- 感動：「泣ける...」
- 笑い：「www」

② 保存したくなる価値
- 後で見返したい情報
- 真似したいテクニック
- 忘れたくないTips

**コメントを増やすテクニック**

① 質問で終わる
- 「みんなはどう思う？」
- 「どっち派？」
- 「経験ある人いる？」

② 議論を呼ぶ
- 「賛否あると思うけど」
- 「正直これってどうなの？」

③ コメント促進フレーズ
```
「同じ経験ある人✋」
「詳しく知りたい人はコメントで」
「[絵文字]で反応して」
「続き知りたい人はコメント」
```

④ ピン留めコメント活用
- 自分でコメントして質問を投げる
- 良いコメントをピン留め

**シェアを増やすテクニック**

① シェアしたくなる要素
- 友達に見せたい面白さ
- 「これ私じゃん」な共感
- 役立つ情報
- 意外な事実

② シェア促進フレーズ
```
「友達にも教えてあげて」
「保存して後で見て」
「これ知らない人多すぎ」
```

**コメント返信戦略**

| コメント種類 | 返信例 |
|-------------|--------|
| 質問 | 「それは〇〇だよ！詳しくは次の動画で」 |
| 褒め | 「ありがとう！嬉しい😊」 |
| 批判 | 「意見ありがとう！参考にする🙏」 |
| 追加要望 | 「リクエストありがとう！作るね」 |

---

📊 **エンゲージメント目標値**
- いいね率：5%以上（視聴数比）
- コメント率：0.5%以上
- シェア率：1%以上
