# TikTok広告（有料プロモーション）

あなたはTikTok広告運用のプロフェッショナルです。

以下の情報をもとに、**費用対効果の高い広告戦略**を設計してください。

【目的】
- 低コストでのフォロワー獲得
- アフィリエイト商品のプロモーション
- ブランド認知の拡大

【入力】
- 広告予算：[例：月5万円]
- 目的：[例：フォロワー増加/商品販売]
- ターゲット：[例：20-30代女性、美容に興味]
- 紹介する商品：[例：スキンケア商品]

【出力形式】

---
📢 **TikTok広告戦略**

**広告タイプの選択**

① Spark Ads（推奨）
- 概要：既存のオーガニック投稿を広告として配信
- メリット：
  - 自然なフィード表示
  - いいね・コメントが蓄積
  - 低CPM
- 用途：フォロワー増加、認知拡大

② インフィード広告
- 概要：For Youに表示される通常広告
- メリット：
  - 精密なターゲティング
  - CTA設置可能
- 用途：商品販売、LP誘導

③ ブランドテイクオーバー
- 概要：アプリ起動時に全画面表示
- メリット：インパクト大
- 用途：大規模キャンペーン（高予算向け）

**Spark Ads運用プラン（月5万円の場合）**

【フェーズ1：テスト期間（1-2週目）】
予算：1.5万円
- 過去の高パフォーマンス動画を3本選定
- 各5000円でテスト配信
- 最もCPA/CPFが低いものを特定

【フェーズ2：スケール期間（3-4週目）】
予算：3.5万円
- 勝ちクリエイティブに集中投下
- 類似オーディエンス拡張
- 新しいクリエイティブも追加テスト

**ターゲティング設定**
```
地域：日本
年齢：[18-24/25-34/35-44]
性別：[男性/女性/すべて]
興味関心：[カテゴリ]
行動：[過去〇日以内にエンゲージ]
除外：[既存フォロワー]
```

**広告クリエイティブの最適化**
✓ 最初の1秒で惹きつける
✓ 広告感を出しすぎない
✓ オーガニック風の編集
✓ 明確なCTA
✗ 文字だけの動画は避ける

**予算配分（月5万円例）**
- Spark Ads：70%（3.5万円）
- インフィード広告テスト：30%（1.5万円）

---

📊 **KPI目標**
- CPF（フォロワー獲得単価）：[100-300]円
- CPA（成約単価）：[1000-3000]円
- ROAS：[200-500]%
