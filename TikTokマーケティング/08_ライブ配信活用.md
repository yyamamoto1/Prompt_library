# TikTokライブ配信活用戦略

あなたはTikTokライブで収益化とフォロワー獲得を実現するプロです。

以下の情報をもとに、**ライブ配信を活用した収益化戦略**を設計してください。

【目的】
- ライブでのギフト収益
- フォロワーとの関係強化
- 商品・サービスへの誘導

【入力】
- フォロワー数：[例：5000人]
- ライブ経験：[例：未経験/月2回]
- 紹介したい商品：[例：オンライン講座]
- 配信可能な時間帯：[例：夜21時〜]

【出力形式】

---
🔴 **ライブ配信戦略**

**ライブの種類と目的**

① 質問回答ライブ
- 頻度：週1-2回
- 内容：フォロワーからの質問に回答
- 収益ポイント：信頼構築→商品への興味喚起

② 雑談ライブ
- 頻度：週1回
- 内容：日常の話、ファンとの交流
- 収益ポイント：ギフト（投げ銭）

③ 企画ライブ
- 頻度：月1-2回
- 内容：〇〇チャレンジ、コラボなど
- 収益ポイント：新規フォロワー獲得

④ 商品紹介ライブ
- 頻度：月1-2回
- 内容：実際に商品を使いながら紹介
- 収益ポイント：アフィリエイト

**ライブ台本テンプレート**

```
【開始前（-5分）】
通知を出す、準備

【オープニング（0-5分）】
・挨拶、名前を呼ぶ
・今日のライブ内容を説明
・「ハートで応援してね」

【メインパート（5-40分）】
・テーマに沿った内容
・コメント拾い
・定期的にギフトへのお礼

【商品紹介（40-50分）】
・自然な流れで紹介
・実際に使って見せる
・「プロフィールにリンクあるよ」

【エンディング（50-60分）】
・まとめ
・次回ライブ告知
・フォロー促し
```

**ギフト収益を上げるコツ**
- ギフトへの反応を大きく
- 定期配信で常連を作る
- ギフトランキング読み上げ
- 目標設定（「〇〇個集まったら〇〇する」）

**最適な配信時間**
| 時間帯 | 視聴者層 | おすすめ |
|--------|----------|----------|
| 7-9時 | 通勤中の社会人 | △ |
| 12-13時 | 昼休み | ○ |
| 18-20時 | 帰宅後 | ◎ |
| 21-24時 | リラックス時間 | ◎ |
