🛒 **TikTokショップ・EC連携戦略**

**販売導線の設計**

【パターン1：TikTokショップ直販】
```
動画視聴
  ↓
商品タグをタップ
  ↓
TikTokショップ内で購入
  ↓
決済完了
```
メリット：離脱率低い、衝動買い促進

【パターン2：プロフィールリンク経由】
```
動画視聴
  ↓
プロフィール訪問
  ↓
リンクタップ
  ↓
外部EC（アフィリエイト）で購入
```
メリット：ASP経由で報酬獲得

【パターン3：ライブショッピング】
```
ライブ視聴
  ↓
商品説明を聞く
  ↓
リアルタイムで質問
  ↓
商品タグから購入
```
メリット：高CVR、信頼性向上

**商品紹介動画の構成**

① 問題提起型（30秒）
```
[0-5秒] 悩み提示：「〇〇で困ってない？」
[5-20秒] 解決策：「これ使ってみて」
[20-28秒] 使用感：「〇〇になった」
[28-30秒] CTA：「リンクはここ👇」
```

② 開封・レビュー型（45秒）
```
[0-3秒] フック：「話題の〇〇買ってみた」
[3-25秒] 開封・使用
[25-40秒] 感想・評価
[40-45秒] CTA：「商品タグからチェック」
```

③ ビフォーアフター型（30秒）
```
[0-10秒] Before
[10-20秒] 商品使用
[20-28秒] After
[28-30秒] CTA
```
